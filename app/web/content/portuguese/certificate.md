# Certificado CS50

Se você enviar e receber uma pontuação de pelo menos 70% em cada um dos projetos deste curso, incluindo seu projeto final, você será elegível para um [Certificado CS50](https://cs50.harvard.edu/certificates) gratuito como o abaixo. Para um [certificado verificado](https://www.edx.org/verified-certificate) da edX, registre-se em [cs50.edx.org/web](http://cs50.edx.org/web). Envie um email para [certificates@cs50.harvard.edu](mailto:certificates@cs50.harvard.edu) com qualquer dúvida.
