Qué hacer

  1. Envía este formulario.

  3. Marca la casilla indicando que deseas conceder acceso al personal del curso a tu envío, y haz clic en Unirse al curso.
  4. Inicia sesión en cs50.dev, que es una versión basada en la nube de Visual Studio Code (VS Code) que te proporciona tu propio "espacio de código" con todo lo necesario para el curso ya instalado. No es necesario descargar ni instalar VS Code o Python en tu propio Mac o PC. Si aún no estás familiarizado, asegúrate de ver el Video Corto de Visual Studio Code para CS50 para obtener una descripción completa.
  5. Una vez que se haya cargado tu espacio de código, cierra cualquier pestaña de Bienvenida que se haya abierto por defecto
  6. Ejecuta update50 en la ventana terminal de tu espacio de código para asegurarte de que está actualizado; si se solicita, haz clic en Reconstruir ahora.
