Définir des types de données personnalisés
• Le mot-clé typedef de C permet de créer un raccourci ou de réécrire le nom des types de données.
• L’idée de base est de d’abord définir un type de manière normale, puis de le donner en alias à quelque chose d’autre.

Définir des types de données personnalisés
struct car
{
int year;
char model[10];
char plate[7];
int odometer;
double engine_size;
};
typedef struct car car_t;
