Ahora el comando que puedo ejecutar aquí va a ser este - Voy a ejecutar Python hola.py. ¿Y por qué es eso? Bueno, resulta que cuando realmente escribo código en un lenguaje como Python, por supuesto, se almacena en ese archivo - hola.py, pero necesito interpretar el código de arriba a abajo, de izquierda a derecha para que la computadora sepa qué hacer. De hecho, al final del día, incluso si no sabes mucho sobre computadoras, probablemente hayas escuchado que las computadoras solo entienden ceros y unos, el llamado sistema binario.

Bueno, si ese es el caso, entonces algo que dice "print" y paréntesis y comillas, no son seguramente ceros y unos. Tenemos que traducirlo de alguna manera en los ceros y unos que la computadora entiende. Ahora, afortunadamente, hay un programa llamado Python para hacer eso. Entonces, Python no solo es un lenguaje en el que vamos a escribir código, también es un programa conocido como un intérprete que instalas de forma gratuita en tu propia computadora o en algún servidor en la nube, y luego puedes ejecutar ese programa, ese intérprete, pasándole como entrada de dato el nombre de tu archivo como el mío aquí, hola.py. Y luego ese programa, ese intérprete, se encargará de leerlo de arriba a abajo, de izquierda a derecha y traducirlo efectivamente en esos ceros y unos que la computadora puede entender.
