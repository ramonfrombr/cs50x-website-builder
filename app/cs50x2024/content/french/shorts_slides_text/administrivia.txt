Administration
● Notation
● Les affectations sont notées sur quatre axes différents.
● Champ d'application
● Avez-vous fait un effort sincère pour résoudre le problème ?
● Exactitude
● Votre code fonctionne-t-il ? Est-il exempt de bogues ?
● Conception
● Dans quelle mesure votre code est-il efficace ? Dans quelle mesure est-il bien écrit ?
● Style
● Dans quelle mesure votre code est-il lisible par l'homme ?
