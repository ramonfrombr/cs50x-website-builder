Vamos a seguir adelante y pasar a otro tipo de dato muy común en Python, y en programación en general, que son los enteros o números enteros, conocidos en Python como INT- I-N-T. Así como STR, S-T-R es una abreviatura de cadena de caracteres en Python, INT es una abreviatura de número entero. Bueno, ¿qué es un número entero? Bueno, al igual que en matemáticas, es un número como -2, -1, 0, 1, 2, y así hasta menos infinito y más infinito.

Pero en un número entero no hay punto decimal. Es solo un número como -2, -1, 0, 1, y así sucesivamente. Eso es un INT. Por supuesto, en el mundo de las matemáticas, hay muchos símbolos que usamos. Y ya hemos visto el símbolo + antes, aunque lo usamos para otro propósito. Pero Python admite estos símbolos y más. Python te permite sumar números juntos +, restar números, multiplicar números y dividir números. Y el único que tal vez se vea un poco extraño o desconocido es este signo de porcentaje, pero no significa "porcentaje" en este contexto. Si usas un solo signo de porcentaje en un programa de Python, en realidad es el llamado operador de módulo, el operador que te permite obtener el resto al dividir un número entre otro. Veremos ejemplos de eso antes de mucho tiempo, pero los primeros cuatro son tal vez bastante, bastante familiares.

Resulta que en Python no necesariamente puedes seguir escribiendo código en un archivo como Hello.py y luego ejecutándolo en una ventana de terminal. Una de las características que a muchas personas les gusta de Python es que admite este llamado modo interactivo. Puedes comenzar a escribir código de Python y ejecutar cada una de esas líneas de manera interactiva, especialmente si no te importa guardar todas tus líneas de código. Solo quieres ejecutar código y obtener algunas respuestas. Por ejemplo, déjame volver a VS Code aquí y cerrar Hello.py y hacer clic en el pequeño triángulo aquí en mi ventana de terminal para hacerla mucho más grande, solo temporalmente por un momento.

Entonces, ahora no estoy creando ningún archivo .py, solo voy a ejecutar Python por sí mismo en mi símbolo de comando. Y verás que cuando hago esto, obtengo una salida que se ve un poco críptica y la fecha y hora en que se actualizó el programa, y así sucesivamente. Pero al final obtengo tres triples corchetes como estos. Esto es el modo interactivo de Python. Así estoy ejecutando el intérprete de Python y cada vez que escribo una línea de código en el intérprete, lo ejecutará de inmediato. No tengo que seguir ejecutando Python una y otra vez. Es como si estuvieras al lado de una persona que habla otro idioma y estás teniendo una conversación con ella de ida y vuelta, todo está sucediendo, la traducción, de inmediato.

¿Qué podría hacer en el modo interactivo? Bueno, podría hacer algo como 1 + 1, Enter. Eso es en realidad código, ¿de acuerdo? Puede que no lo pienses como código, pero si tienes algo de aritmética y conoces los números y el +, eso es código Python válido. Y realmente puedes usar Python como una calculadora sofisticada. Pero también podría hacer otras cosas. Si quiero imprimir para mí mismo "Hola, mundo", también puedo imprimir esa línea de código así - "Hola, mundo". Así que es interactivo en el sentido de que en cuanto ejecutas una línea de código, ¡boom, ves el resultado!

Generalmente no vamos a hacer eso, porque al menos al enseñar un lenguaje, tendemos a querer hacer las cosas paso a paso y queremos que puedas ver de dónde venimos. Y queremos poder probar las cosas una y otra vez, especialmente si cometemos errores. Pero debes saber que esto es realmente una característica de Python, este llamado modo interactivo. Pero enfoquémonos por un momento no solo en esa interactividad, sino realmente en el hecho de que Python aparentemente admite enteros y matemáticas y algunas de esas operaciones básicas. Y veamos si no podemos crear nuestra propia pequeña calculadora.
