Donc, dans cet environnement de programmation, je vais aller de l'avant et cliquer sur ce signe plus en haut pour un nouveau fichier. Par convention, je vais aller de l'avant et nommer mon fichier quelque chose comme bonjour.c. Bonjour simplement parce que je veux que ce programme dise bonjour, mais je pourrais l'appeler comme je le souhaite. Et point c parce que la convention, dans ce langage de programmation C, est de nommer ses fichiers qui contiennent leur code avec littéralement point c, un peu comme vous avez point mp4 pour des vidéos, point gif pour les fichiers graphiques, etc. Donc, je vais aller de l'avant et cliquer sur Nouveau fichier. Je vais nommer mon fichier.

C'est ici que je peux écrire du code, dans un onglet appelé bonjour.c. Il va automatiquement numéroter mes lignes pour moi. Maintenant, même si le code que je suis sur le point d'écrire est, franchement, plutôt énigmatique – include stdio.h, int main void. Et puis ici, je vais faire printf, guillemets, "bonjou, le monde" point-virgule.

Je viens d'écrire mon premier programme en C. Il y a en tout six lignes de code. Et vous verrez que j'ai écrit include stdio.h, quoique cela signifie, int main void, quoique cela signifie. Et puis, vraiment, l'essence de ce programme est sur la ligne 5 – imprimez, c'est ça que printf signifie, "bonjour, le monde." Donc, souvenez-vous que int main void est similaire dans l'esprit à quand drapeau vert est cliqué. Et printf est similaire au bloc dire. Et nous avons simplement besoin de cette ligne de code supérieure, car l'ordinateur ne comprendra pas printf à moins que je lui dise d'inclure le fichier stdio.h, avec la première ligne.
