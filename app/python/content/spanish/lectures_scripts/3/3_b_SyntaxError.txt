Así que el problema con los errores de sintaxis es que depende completamente de ti resolverlos. Un error de sintaxis es un problema que tienes que corregir en tu código desde el principio. No puedes esperar que se resuelva solo o esperar que otras partes de tu código lo detecten por ti. Los errores de sintaxis simplemente deben corregirse.

Pero hay muchos otros tipos de errores en Python que podrían describirse como errores de tiempo de ejecución, que ocurren mientras tu código se está ejecutando.
