# Introduction to Algorithms

Algorithms are a set of rules or steps to solve a problem. They are used to perform calculations, data processing, automated reasoning, and other tasks in computer science. 

## Characteristics of Algorithms

- Input: Algorithms take zero or more inputs.
- Output: Algorithms produce at least one output.
- Definiteness: Each step in an algorithm must be precisely defined.
- Finiteness: Algorithms must terminate after a finite number of steps.
- Effectiveness: Every operation in an algorithm must be clear and understandable.

## Example: Bubble Sort

Bubble Sort is a simple algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted.

# Introdução aos Algoritmos

Algoritmos são um conjunto de regras ou passos para resolver um problema. Eles são usados para realizar cálculos, processamento de dados, raciocínio automatizado e outras tarefas na ciência da computação.

## Características dos Algoritmos

- Entrada: Algoritmos recebem zero ou mais entradas.
- Saída: Algoritmos produzem pelo menos uma saída.
- Definição: Cada passo em um algoritmo deve ser precisamente definido.
- Finitude: Algoritmos devem terminar após um número finito de etapas.
- Eficácia: Cada operação em um algoritmo deve ser clara e compreensível.

## Exemplo: Bubble Sort

Bubble Sort é um algoritmo simples que passa repetidamente pela lista, compara elementos adjacentes e os troca se estiverem na ordem errada. A passagem pela lista é repetida até que nenhuma troca seja necessária, o que indica que a lista está ordenada.
