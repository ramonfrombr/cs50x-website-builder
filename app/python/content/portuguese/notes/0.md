## Criando código com Python <a id="creating-code-with-python"></a>

- O VS Code é um tipo especial de editor de texto chamado de compilador. Na parte superior, você encontrará um editor de texto. Na parte inferior, você verá um terminal onde pode executar comandos.
- No terminal, você pode executar `code hello.py` para começar a programar.
- No editor de texto acima, você pode digitar `print("olá, mundo")`. Este é um programa canônico famoso que quase todos os programadores escrevem durante seu processo de aprendizado.
- Na janela do terminal, você pode executar comandos. Para executar este programa, você precisa mover seu cursor para a parte inferior da tela, clicando na janela do terminal. Agora, você pode digitar um segundo comando na janela do terminal. Ao lado do símbolo de dólar, digite `python hello.py` e pressione a tecla enter no seu teclado.
- Lembre-se de que os computadores realmente só entendem zeros e uns. Portanto, quando você executa `python hello.py`, o Python irá interpretar o texto que você criou em `hello.py` e traduzi-lo para zeros e uns que o computador pode entender.
- O resultado da execução do programa `python hello.py` é `olá, mundo`.
- Parabéns! Você acabou de criar seu primeiro programa.

## Funções <a id="funcoes"></a>

- Funções são verbos ou ações que o computador ou linguagem de programação já sabe como executar.
- No seu programa `hello.py`, a função `print` sabe como imprimir na janela do terminal.
- A função `print` recebe argumentos. Neste caso, `"hello, world"` são os argumentos que a função `print` recebe.

## Erros <a id="erros"></a>

- Erros são uma parte natural da programação. São erros, problemas para você resolver! Não se desanime! Isso faz parte do processo de se tornar um ótimo programador.
- Imagine no nosso programa `hello.py` que acidentalmente digitamos `print("hello, world"` repare que esquecemos o `)` final necessário pelo compilador. Se eu fizer esse erro de propósito, você verá que o compilador emitirá um erro na janela do terminal!
- Frequentemente, as mensagens de erro irão informar sobre seus erros e fornecer indícios sobre como corrigi-los. No entanto, muitas vezes o compilador não será tão amigável.

## Melhorando o Seu Primeiro Programa em Python <a id="melhorando-o-seu-primeiro-programa-em-python"></a>

- Podemos personalizar o seu primeiro programa em Python.
- No nosso editor de texto no `hello.py` podemos adicionar outra função. `input` é uma função que recebe uma mensagem como argumento. Podemos editar o nosso código para ficar assim:

      input("Qual é o seu nome? ")
      print("olá, mundo")

- No entanto, essa edição por si só não permitirá que o programa imprima o que o usuário digitou. Para isso, precisaremos te apresentar às variáveis.

### Variáveis <a id="variaveis"></a>

- Uma variável é apenas um recipiente para um valor no seu próprio programa.
- No seu programa, você pode introduzir a sua própria variável editando-o da seguinte forma:

      nome = input("Qual é o seu nome? ")
      print("olá, mundo")

  Note que esse sinal de `=` no meio de `nome = input("Qual é o seu nome? ")` tem um papel especial na programação. Esse sinal de igual literalmente atribui o valor à direita para o valor à esquerda. Portanto, o valor retornado por `input("Qual é o seu nome? ")` é atribuído à variável `nome`.

- Se você editar o seu código como segue, você verá um erro:

      nome = input("Qual é o seu nome? ")
      print("olá, nome")

- O programa irá retornar `olá, nome` na janela do terminal, independentemente do que o usuário digitar.
- Editando ainda mais o nosso código, você poderia digitar:

      nome = input("Qual é o seu nome? ")
      print("olá,")
      print(nome)

- O resultado na janela do terminal seria:

      Qual é o seu nome? David
      olá,
      David

- Estamos chegando mais perto do resultado que pretendemos!
- Você pode aprender mais na documentação do Python sobre [tipos de dados](https://docs.python.org/3/library/datatypes.html).

### Comentários <a id="comentarios"></a>

- Comentários são uma forma dos programadores acompanharem o que estão fazendo em seus programas e até mesmo informarem aos outros sobre suas intenções para um bloco de código. Resumindo, eles são notas para você mesmo e para os outros que irão ver o seu código!
- Você pode adicionar comentários ao seu programa para poder ver o que ele está fazendo. Você poderia editar o seu código da seguinte forma:

      # Pergunte ao usuário o seu nome
      nome = input("Qual é o seu nome? ")
      print("olá,")
      print(nome)

- Comentários também podem servir como uma lista de tarefas para você.

### Pseudocódigo <a id="pseudocodigo"></a>

- Pseudocódigo é um tipo de comentário importante que se torna um tipo especial de lista de tarefas, especialmente quando você não entende como realizar uma tarefa de programação. Por exemplo, no seu código, você poderia editar o seu código para ficar assim:

      # Pergunte ao usuário o seu nome
      nome = input("Qual é o seu nome? ")

      # Imprima olá
      print("olá,")

      # Imprima o nome digitado
      print(nome)

## Melhorando ainda mais o seu primeiro programa Python <a id="melhorando-ainda-mais-o-seu-primeiro-programa-python"></a>

- Podemos editar ainda mais nosso código da seguinte maneira:

      # Pergunte ao usuário o seu nome
      name = input("Qual é o seu nome? ")

      # Imprime olá e o nome digitado
      print("olá, " + name)

- Acontece que algumas funções recebem vários argumentos.
- Podemos usar uma vírgula `,` para passar múltiplos argumentos editando nosso código da seguinte maneira:

      # Pergunte ao usuário o seu nome
      name = input("Qual é o seu nome? ")

      # Imprime olá e o nome digitado
      print("olá,", name)

  A saída no terminal, se digitarmos "David", seria `olá, David`. Sucesso.

## Strings e Parâmetros <a id="strings-e-parametros"></a>

- Uma string, conhecida como `str` em Python, é uma sequência de texto.
- Voltando um pouco no nosso código para o seguinte, houve um efeito visual de ter o resultado aparecendo em várias linhas:

      # Pergunte ao usuário seu nome
      nome = input("Qual é o seu nome? ")
      print("olá,")
      print(nome)

- Funções recebem argumentos que influenciam seu comportamento. Se olharmos a documentação para [`print`](https://docs.python.org/3/library/functions.html#print), você perceberá que podemos aprender muito sobre os argumentos que a função print recebe.
- Olhando essa documentação, você aprenderá que a função print automaticamente inclui um trecho de código `end='\n'`. Esse `\n` indica que a função print automaticamente criará uma quebra de linha quando executada. A função print recebe um argumento chamado `end` e o valor padrão é criar uma nova linha.
- No entanto, tecnicamente podemos fornecer um argumento para `end` nós mesmos para que uma nova linha não seja criada!
- Podemos modificar nosso código da seguinte forma:

      # Pergunte ao usuário seu nome
      nome = input("Qual é o seu nome? ")
      print("olá,", end="")
      print(nome)

  Ao fornecer `end=""`, estamos sobrescrevendo o valor padrão de `end` para que ele nunca crie uma nova linha após essa primeira instrução de impressão. Fornecendo o nome como "David", a saída na janela do terminal será `olá, David`.

- Parâmetros, portanto, são argumentos que podem ser recebidos por uma função.
- Você pode aprender mais na documentação do Python sobre [`print`](https://docs.python.org/3/library/functions.html#print).

### Um pequeno problema com aspas <a id="um-pequeno-problema-com-aspas"></a>

- Observe como adicionar aspas como parte da sua string é desafiador.
- `print("olá,"amigo"")` não funcionará e o compilador emitirá um erro.
- Geralmente, existem duas abordagens para corrigir isso. Primeiro, você poderia simplesmente alterar as aspas para aspas simples.
- Outra abordagem, mais comumente usada, seria codificar como `print("olá, \"amigo\"")`. As barras invertidas dizem ao compilador que o caractere seguinte deve ser considerado uma aspa na string e evitam um erro de compilação.

## Formatação de Strings <a id="formatacao-de-strings"></a>

- Provavelmente, a maneira mais elegante de usar strings seria da seguinte forma:

      # Peça ao usuário para inserir o nome
      nome = input("Qual é o seu nome? ")
      print(f"olá, {nome}")

  Observe o `f` em `print(f"olá, {nome}")`. Esse `f` é um indicador especial para o Python tratar essa string de maneira especial, diferente das abordagens anteriores que foram ilustradas nesta aula. Espere utilizar este estilo de strings com frequência neste curso.

## Mais sobre Strings <a id="mais-sobre-strings"></a>

- Você nunca deve esperar que seu usuário coopere como pretendido. Portanto, você precisará garantir que a entrada do usuário seja corrigida ou verificada.
- Acontece que nas strings incorporadas está a capacidade de remover espaços em branco de uma string.
- Utilizando o método `strip` em `name` como `name = name.strip()`, irá remover todos os espaços em branco à esquerda e à direita da entrada do usuário. Você pode modificar seu código para ficar assim:

      # Pergunte ao usuário qual é o nome dele
      name = input("Qual é o seu nome? ")

      # Remova os espaços em branco da string
      name = name.strip()

      # Imprima a saída
      print(f"olá, {name}")

  Executando este programa, independentemente da quantidade de espaços que você digitar antes ou depois do nome, ele removerá todos os espaços em branco.

- Usando o método `title`, ele colocará o nome do usuário em caixa alta:

      # Pergunte ao usuário qual é o nome dele
      name = input("Qual é o seu nome? ")

      # Remova os espaços em branco da string
      name = name.strip()

      # Coloque a primeira letra de cada palavra em caixa alta
      name = name.title()

      # Imprima a saída
      print(f"olá, {name}")

- Neste ponto, você pode estar bem cansado de digitar `python` repetidamente na janela do terminal. Você pode usar a seta para cima do seu teclado para relembrar os comandos mais recentes do terminal que você fez.
- Observe que você pode modificar seu código para ser mais eficiente:

      # Pergunte ao usuário qual é o nome dele
      name = input("Qual é o seu nome? ")

      # Remova os espaços em branco da string e coloque a primeira letra de cada palavra em caixa alta
      name = name.strip().title()

      # Imprima a saída
      print(f"olá, {name}")

  Isso cria o mesmo resultado que seu código anterior.

- Podemos ir ainda mais longe!

      # Pergunte ao usuário qual é o nome dele, remova os espaços em branco da string e coloque a primeira letra de cada palavra em caixa alta
      name = input("Qual é o seu nome? ").strip().title()

      # Imprima a saída
      print(f"olá, {name}")

- Você pode aprender mais sobre strings na documentação do Python em [`str`](https://docs.python.org/3/library/stdtypes.html#str)

## Inteiros ou int <a id="inteiros-ou-int"></a>

- Em Python, um número inteiro é chamado de `int`.
- No mundo da matemática, estamos familiarizados com os operadores +, -, \*, / e %. Este último operador `%` ou operador de módulo pode não ser muito familiar para você.
- Você não precisa usar a janela do editor de texto no seu compilador para executar código Python. Na sua terminal, você pode executar apenas `python`. Você verá `>>>` na janela do terminal e poderá executar código ao vivo e interativo. Você pode digitar `1+1` e ele executará o cálculo. Este modo não será comumente usado durante este curso.
- Abrindo o VS Code novamente, podemos digitar `code calculator.py` no terminal. Isso criará um novo arquivo no qual criaremos nossa própria calculadora.
- Primeiro, podemos declarar algumas variáveis.

       x = 1
       y = 2

       z = x + y

       print(z)

  Naturalmente, quando executamos `python calculator.py`, obtemos o resultado na janela do terminal como `3`. Podemos tornar isso mais interativo usando a função `input`.

       x = input("Qual é o valor de x? ")
       y = input("Qual é o valor de y? ")

       z = x + y

       print(z)

- Ao executar esse programa, descobrimos que a saída está incorreta, retornando `12`. Por que isso pode estar acontecendo?
- Anteriormente, vimos como o sinal de `+` concatena duas strings. Como a entrada do teclado no seu computador é recebida pelo compilador como texto, ela é tratada como uma string. Portanto, precisamos converter essa entrada de uma string para um inteiro. Podemos fazer isso da seguinte forma:

       x = input("Qual é o valor de x? ")
       y = input("Qual é o valor de y? ")

       z = int(x) + int(y)

       print(z)

  Agora o resultado está correto. O uso de `int(x)` é chamado de "casting" ou conversão de tipo, onde um valor é temporariamente alterado de um tipo de variável (neste caso, uma string) para outro (neste caso, um inteiro).

- Podemos melhorar ainda mais nosso programa da seguinte forma:

       x = int(input("Qual é o valor de x? "))
       y = int(input("Qual é o valor de y? "))

       print(x + y)

  Isso ilustra que você pode executar funções em funções. A função interna é executada primeiro e, em seguida, a externa é executada. Primeiro, a função `input` é executada e, em seguida, a função `int`.

- Você pode aprender mais na documentação do Python sobre [`int`](https://docs.python.org/3/library/functions.html?highlight=float#int).

## Readabilidade Vence <a id="readability-wins"></a>

- Ao decidir a abordagem para uma tarefa de codificação, lembre-se de que é possível argumentar razoavelmente por várias abordagens para o mesmo problema.
- Independentemente da abordagem que você escolher para uma tarefa de programação, lembre-se de que seu código deve ser legível. Você deve usar comentários para dar a si mesmo e aos outros pistas sobre o que seu código está fazendo. Além disso, você deve criar código de maneira legível.

# Noções básicas sobre números de ponto flutuante <a id="float-basics"></a>

- Um valor de ponto flutuante é um número real que possui um ponto decimal, como `0.52`.
- Você pode alterar seu código para aceitar números de ponto flutuante da seguinte maneira:

      x = float(input("Qual é o valor de x? "))
      y = float(input("Qual é o valor de y? "))

      print(x + y)

  Essa alteração permite que o usuário insira `1.2` e `3.4` para obter um total de `4.6`.

- Vamos supor, no entanto, que você queira arredondar o total para o inteiro mais próximo. Ao consultar a documentação do Python para a função `round`, você verá que os argumentos disponíveis são `round(number[n, ndigits])`. Esses colchetes indicam que algo opcional pode ser especificado pelo programador. Portanto, você pode usar `round(n)` para arredondar um dígito para o inteiro mais próximo. Alternativamente, você pode codificar da seguinte maneira:

      # Obter a entrada do usuário
      x = float(input("Qual é o valor de x? "))
      y = float(input("Qual é o valor de y? "))

      # Criar um resultado arredondado
      z = round(x + y)

      # Imprimir o resultado
      print(z)

  A saída será arredondada para o inteiro mais próximo.

- E se quiséssemos formatar a saída de números grandes? Por exemplo, em vez de exibir `1000`, você pode desejar ver `1.000`. Você pode modificar o código da seguinte maneira:

      # Obter a entrada do usuário
      x = float(input("Qual é o valor de x? "))
      y = float(input("Qual é o valor de y? "))

      # Criar um resultado arredondado
      z = round(x + y)

      # Imprimir o resultado formatado
      print(f"{z:,}")

  Embora um tanto criptico, `print(f"{z:,}")` permite que a saída `z` inclua vírgulas, para que o resultado possa parecer `1.000` ou `2.500`.

## Mais sobre Floats <a id="mais-sobre-floats"></a>

- Como podemos arredondar valores de ponto flutuante? Primeiro, modifique seu código da seguinte maneira:

      # Obtenha a entrada do usuário
      x = float(input("Qual é o valor de x? "))
      y = float(input("Qual é o valor de y? "))

      # Calcule o resultado
      z = x / y

      # Imprima o resultado
      print(z)

  Ao inserir `2` para x e `3` para y, o resultado z é `0.6666666666`, aparentemente continuando infinitamente, como poderíamos esperar.

- Vamos imaginar que queremos arredondar esse valor para baixo. Podemos modificar nosso código da seguinte maneira:

      # Obtenha a entrada do usuário
      x = float(input("Qual é o valor de x? "))
      y = float(input("Qual é o valor de y? "))

      # Calcule o resultado e arredonde
      z = round(x / y, 2)

      # Imprima o resultado
      print(z)

  Como poderíamos esperar, isso arredondará o resultado para as duas casas decimais mais próximas.

- Também podemos usar `fstring` para formatar a saída da seguinte maneira:

      # Obtenha a entrada do usuário
      x = float(input("Qual é o valor de x? "))
      y = float(input("Qual é o valor de y? "))

      # Calcule o resultado
      z = x / y

      # Imprima o resultado
      print(f"{z:.2f}")

  Esse código enigmático `fstring` exibe o mesmo resultado que nossa estratégia anterior de arredondamento.

- Você pode aprender mais na documentação do Python sobre [`float`](https://docs.python.org/3/library/functions.html?highlight=float#float).

## Definição <a id="def"></a>

- Não seria bom criarmos nossas próprias funções?
- Vamos trazer de volta nosso código final do `hello.py` digitando `code hello.py` na janela do terminal. Seu código inicial deve ser assim:

      # Pergunte ao usuário o nome dele, remova os espaços em branco da str e capitalize a primeira letra de cada palavra
      name = input("Qual é o seu nome? ").strip().title()

      # Imprima o resultado
      print(f"olá, {name}")

  Podemos melhorar nosso código para criar nossa própria função especial que diz "olá" para nós!

- Apagando todo nosso código no editor de texto, vamos começar do zero:

      name = input("Qual é o seu nome? ")
      hello()
      print(name)

  Tentando executar este código, o compilador mostrará um erro. Afinal, não há uma função definida chamada `hello`.

- Podemos criar nossa própria função chamada `hello` da seguinte maneira:

      def hello():
          print("olá")


      name = input("Qual é o seu nome? ")
      hello()
      print(name)

  Note que tudo abaixo de `def hello()` está indentado. O Python é uma linguagem indentada. Ele usa a indentação para entender o que faz parte da função acima. Portanto, tudo na função `hello` deve estar indentado. Quando algo não está indentado, é tratado como se não estivesse dentro da função `hello`. Ao executar `python hello.py` na janela do terminal, você verá que a saída não é exatamente como você gostaria.

- Podemos melhorar ainda mais nosso código:

      # Crie nossa própria função
      def hello(to):
          print("olá,", to)


      # Saída usando nossa própria função
      name = input("Qual é o seu nome? ")
      hello(name)

  Aqui, nas primeiras linhas, você está criando sua função `hello`. Desta vez, entretanto, você está informando ao compilador que esta função recebe um único parâmetro: uma variável chamada `to`. Portanto, quando você chama `hello(name)`, o computador passa `name` para a função `hello` como `to`. É assim que passamos valores para funções. Muito útil! Ao executar `python hello.py` na janela do terminal, você verá que a saída está muito mais próxima do que foi apresentado anteriormente nesta aula.

- Podemos alterar nosso código para adicionar um valor padrão a `hello`:

      # Crie nossa própria função
      def hello(to="mundo"):
          print("olá,", to)


      # Saída usando nossa própria função
      name = input("Qual é o seu nome? ")
      hello(name)

      # Saída sem passar os argumentos esperados
      hello()

  Teste o código você mesmo. Repare como o primeiro `hello` irá se comportar como você espera, e o segundo hello, que não recebe um valor, irá, por padrão, exibir `olá, mundo`.

- Não precisamos ter nossa função no início do programa. Podemos movê-la para baixo, mas precisamos informar ao compilador que temos uma função `main` e uma função `hello` separada.

      def main():

          # Saída usando nossa própria função
          name = input("Qual é o seu nome? ")
          hello(name)

          # Saída sem passar os argumentos esperados
          hello()


      # Crie nossa própria função
      def hello(to="mundo"):
          print("olá,", to)

  Isso por si só, entretanto, irá gerar um erro. Se executarmos `python hello.py`, nada acontece! O motivo para isso é que nada neste código está realmente chamando a função `main` e dando vida ao nosso programa.

- A seguinte modificação muito pequena irá chamar a função `main` e restaurar nosso programa ao funcionamento normal:

      def main():

          # Saída usando nossa própria função
          name = input("Qual é o seu nome? ")
          hello(name)

          # Saída sem passar os argumentos esperados
          hello()


      # Crie nossa própria função
      def hello(to="mundo"):
          print("olá,", to)


      main()

## Retornando Valores <a id="retornando-valores"></a>

- Você pode imaginar muitos cenários em que não apenas deseja que uma função execute uma ação, mas também retorne um valor de volta para a função principal. Por exemplo, em vez de simplesmente imprimir o cálculo de `x + y`, você pode querer que uma função retorne o valor desse cálculo de volta para outra parte do seu programa. Chamamos esse "retorno" de um valor de retorno.
- Voltando para o nosso código `calculator.py` digitando `code calculator.py`. Apague todo o código lá. Refaça o código da seguinte forma:

      def main():
          x = int(input("Qual é o valor de x? "))
          print("x ao quadrado é", square(x))


      def square(n):
          return n * n


      main()

  Efetivamente, `x` é passado para `square`. Em seguida, o cálculo de `x * x` é retornado para a função principal.
