## Loops <a id="loops"></a>

- Em essência, loops são uma forma de executar algo repetidamente.
- Comece digitando `code cat.py` na janela do terminal.
- No editor de texto, comece com o seguinte código:

      print("meow")
      print("meow")
      print("meow")

  Ao executar esse código digitando `python cat.py`, você notará que o programa mia três vezes.

- Ao se desenvolver como programador, é importante considerar como melhorar áreas do código onde você digita a mesma coisa repetidamente. Imagine se você quisesse "miar" 500 vezes. Seria lógico digitar a mesma expressão `print("meow")` repetidamente?
- Os loops permitem que você crie um bloco de código que é executado repetidamente.

## Loops While <a id="while-loops"></a>

- O loop `while` é praticamente universal em todas as linguagens de programação.
- Esse tipo de loop repetirá um bloco de código várias vezes.
- No editor de texto, edite seu código da seguinte forma:

      i = 3
      while i != 0:
          print("meow")

  Perceba que mesmo que esse código execute `print("meow")` várias vezes, ele nunca irá parar! Ele continuará em loop infinito. Os loops `while` funcionam perguntando repetidamente se a condição do loop foi cumprida. Neste caso, o compilador está perguntando: "o valor de `i` é diferente de zero?". Quando você fica preso em um loop infinito, você pode pressionar `control-c` no teclado para sair do loop.

- Para corrigir esse loop infinito, podemos editar nosso código da seguinte forma:

      i = 3
      while i != 0:
        print("meow")
        i = i - 1

  Agora, nosso código é executado corretamente, diminuindo o valor de `i` em `1` a cada "iteração" do loop. O termo iteração tem um significado especial na programação. Por iteração, queremos dizer um ciclo completo do loop. A primeira iteração é a iteração "0". A segunda é a iteração "1". Na programação, contamos a partir de 0, depois 1, depois 2.

- Podemos melhorar ainda mais nosso código da seguinte forma:

        i = 1
        while i <= 3:
            print("meow")
            i = i + 1

  Perceba que quando escrevemos `i = i + 1`, estamos atribuindo o valor de `i` da direita para a esquerda. Acima, estamos começando `i` com 1, como a maioria das pessoas conta (1, 2, 3). Se você executar o código acima, verá que ele emite três "meow". É uma boa prática na programação começar a contagem a partir de zero.

- Podemos melhorar ainda mais nosso código começando a contagem a partir de zero:

      i = 0
      while i < 3:
          print("miau")
          i += 1

  Observe que ao mudarmos o operador para `i < 3`, nosso código funciona como pretendido. Começamos contando com 0 e ele itera através do loop três vezes, produzindo três "meow". Além disso, observe como `i += 1` é igual a `i = i + 1`.

- Neste ponto, nosso código é ilustrado da seguinte forma: <pre class="mermaid">
flowchart TD
A([início]) --> B[i = 1]
B --> C{i <= 3}
C -- Verdadeiro --> D["#quot;miau#quot;"]
D --> E[i = i + 1]
E --> C
C -- Falso --> F([fim])
</pre>
  Observe como nosso loop conta `i` até, mas não até chegar em `3`.

## Loops For <a id="for-loops"></a>

- Um laço `for` é um tipo diferente de laço.
- Para entender melhor um laço `for`, é melhor começar falando sobre um novo tipo de variável chamada `lista` em Python. Assim como em outras áreas de nossas vidas, podemos ter uma lista de supermercado, uma lista de tarefas, etc.
- Um laço `for` itera através de uma `lista` de itens. Por exemplo, na janela do editor de texto, modifique seu código `cat.py` da seguinte forma:

      for i in [0, 1, 2]:
          print("miau")

  Note como esse código é mais limpo em comparação com seu código anterior usando o laço `while`. Neste código, `i` começa com `0`, mia, `i` é atribuído `1`, mia, e, finalmente, `i` é atribuído `2`, mia, e então termina.

- Embora este código atinja o que queremos, há algumas possibilidades de melhorar nosso código para casos extremos. À primeira vista, nosso código parece ótimo. No entanto, e se você quiser iterar até um milhão? É melhor criar um código que possa lidar com esses casos extremos. Portanto, podemos melhorar nosso código da seguinte forma:

      for i in range(3):
          print("miau")

  Note como `range(3)` retorna automaticamente três valores (`0`, `1` e `2`). Este código será executado e produzirá o efeito pretendido, miando três vezes.

- Nosso código pode ser ainda mais melhorado. Note como nunca usamos `i` explicitamente em nosso código. Ou seja, embora o Python precise do `i` como um local para armazenar o número da iteração do laço, nunca o usamos para nenhum outro propósito. Em Python, se essa variável não tiver nenhum outro significado em nosso código, podemos simplesmente representar essa variável como um sublinhado `_`. Portanto, você pode modificar seu código da seguinte forma:

      para _ em range(3):
          print("miau")

  Note como alterar o `i` para `_` não tem impacto algum no funcionamento do nosso programa.

- Nosso código pode ser ainda mais melhorado. Para expandir sua mente para as possibilidades dentro do Python, considere o seguinte código:

      print("miau" * 3)

  Note como ele irá miar três vezes, mas o programa produzirá `miaumiaumiau` como resultado. Considere: Como você poderia criar uma quebra de linha no final de cada miau?

- Na verdade, você pode editar seu código da seguinte forma:

      print("miau\n" * 3, end="")

  Note como esse código produz três miaus, cada um em uma linha separada. Ao adicionar `end=""` e `\n`, dizemos ao compilador para adicionar uma quebra de linha ao final de cada miau.

## Melhorando com Entrada do Usuário <a id="melhorando-com-entrada-do-usuário"></a>

- Talvez queiramos obter entrada do usuário. Podemos usar loops como forma de validar a entrada do usuário.
- Um paradigma comum em Python é utilizar um loop `while` para validar a entrada do usuário.
- Por exemplo, vamos tentar solicitar ao usuário um número maior ou igual a 0:

       while True:
           n = int(input("Qual é o valor de n? "))
           if n < 0:
               continue
           else:
               break

- Note que introduzimos duas palavras-chave novas em Python, `continue` e `break`. `continue` diz explicitamente ao Python para ir para a próxima iteração do loop. `break`, por outro lado, diz ao Python para "sair" do loop antecipadamente, antes de terminar todas as iterações. Neste caso, iremos `continuar` para a próxima iteração do loop quando `n` for menor que 0, fazendo o usuário ser solicitado novamente com "Qual é o valor de n?". No entanto, se `n` for maior ou igual a 0, iremos `sair` do loop e permitir que o restante do nosso programa seja executado.
- Acontece que a palavra-chave `continue` é redundante nesse caso. Podemos melhorar nosso código da seguinte maneira:

       while True:
           n = int(input("Qual é o valor de n? "))
           if n > 0:
               break

       for _ in range(n):
           print("miado")

  Note como esse loop while sempre será executado (para sempre) até que `n` seja maior que `0`. Quando `n` é maior que `0`, o loop é interrompido.

- Levando em consideração nosso aprendizado anterior, podemos usar funções para melhorar ainda mais nosso código:

       def main():
           miar(obter_numero())


       def obter_numero():
           while True:
               n = int(input("Qual é o valor de n? "))
               if n > 1:
                   return n


       def miar(n):
           for _ in range(n):
               print("miado")


       main()

  Note como não apenas alteramos o seu código para operar em várias funções, mas também utilizamos a instrução `return` para `retornar` o valor de `n` de volta para a função `main`.

## Mais Sobre Listas <a id="mais-sobre-listas"></a>

- Considere o mundo de Hogwarts do famoso universo Harry Potter.
- No terminal, digite `code hogwarts.py`.
- No editor de texto, escreva o código abaixo:

      alunos = ["Hermione", "Harry", "Rony"]

      print(alunos[0])
      print(alunos[1])
      print(alunos[2])

  Observe como temos uma `lista` de alunos com seus nomes como mostrado acima. Em seguida, imprimimos o aluno que está na posição 0, "Hermione". Cada um dos outros alunos também é impresso.

- Assim como ilustramos anteriormente, podemos usar um loop para iterar sobre a lista. Você pode melhorar seu código da seguinte forma:

      alunos = ["Hermione", "Harry", "Rony"]

      for aluno in alunos:
          print(aluno)

  Observe que para cada `aluno` na lista `alunos`, ele imprimirá o aluno como pretendido. Você pode se perguntar por que não usamos a designação `_` como discutimos antes. Optamos por não fazer isso porque `aluno` é explicitamente usado em nosso código.

- Você pode aprender mais na documentação do Python sobre [listas](https://docs.python.org/3/tutorial/datastructures.html#more-on-lists).

## Comprimento <a id="length"></a>

- Podemos usar `len` como uma forma de verificar o comprimento da lista chamada `students`.
- Imagine que você não deseja apenas imprimir o nome do aluno, mas também a posição dele na lista. Para realizar isso, você pode editar seu código da seguinte forma:

      students = ["Hermoine", "Harry", "Ron"]

      for i in range(len(students)):
          print(i + 1, students[i])

  Observe como a execução desse código resulta não apenas em obter a posição de cada aluno mais um usando `i + 1`, mas também imprime o nome de cada aluno. `len` permite que você veja dinamicamente o tamanho da lista dos alunos, independentemente de seu crescimento.

- Você pode aprender mais na documentação do Python sobre [len](https://docs.python.org/3/library/functions.html?highlight=len#len).

## Dicionários <a id="dicionarios"></a>

- `dict`s ou dicionários são uma estrutura de dados que permitem associar chaves a valores.
- Enquanto uma `list` é uma lista de múltiplos valores, um `dict` associa uma chave a um valor.
- Considerando as casas de Hogwarts, podemos atribuir estudantes específicos a casas específicas.

  ![Nomes de Harry Potter.](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide17.png "Nomes de Harry Potter.")

- Poderíamos usar apenas `list`s para realizar isso:

      students = ["Hermoine", "Harry", "Ron", "Draco"]
      houses = ["Gryffindor", "Gryffindor", "Gryffindor", "Slytherin"]

  Observe que podemos garantir que sempre manteremos essas listas em ordem. A pessoa na primeira posição de `students` está associada à casa na primeira posição da lista `houses`, e assim por diante. No entanto, isso pode se tornar bastante complicado à medida que nossas listas crescem!

- Podemos melhorar nosso código usando um `dict` da seguinte maneira:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      print(students["Hermoine"])
      print(students["Harry"])
      print(students["Ron"])
      print(students["Draco"])

  Observe como usamos chaves `{}` para criar um dicionário. Enquanto as `list`s usam números para percorrer a lista, os `dict`s nos permitem usar palavras.

- Execute seu código e verifique se sua saída é a seguinte:

      $ python hogwarts.py
      Gryffindor
      Gryffindor
      Gryffindor
      Slytherin

- Podemos melhorar nosso código da seguinte maneira:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      for student in students:
          print(student)

  Observe que, ao executar esse código, o loop for iterará apenas pelas chaves, resultando em uma lista com os nomes dos estudantes. Como podemos imprimir tanto os valores quanto as chaves?

- Modifique seu código da seguinte maneira:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      for student in students:
          print(student, students[student])

  Observe como `students[student]` irá até a chave de cada estudante e encontrará o valor da casa deles. Execute seu código e observe como a saída fica um pouco bagunçada.

- Podemos melhorar a função de impressão ao aprimorar nosso código da seguinte maneira:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      for student in students:
          print(student, students[student], sep=", ")

  Observe como isso cria uma separação limpa de `,` entre cada item impresso.

- Se você executar `python hogwarts.py`, verá o seguinte:

      $ python hogwarts.py
      Hermoine, Gryffindor
      Harry, Gryffindor
      Ron, Gryffindor
      Draco, Slytherin

- E se tivermos mais informações sobre nossos estudantes? Como podemos associar mais dados a cada um dos estudantes?

  ![Nomes de Harry Potter.](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide18.png "Nomes de Harry Potter.")

- Você pode imaginar querendo ter muitos dados associados a várias chaves. Melhore seu código da seguinte maneira:

      students = [
          {"name": "Hermoine", "house": "Gryffindor", "patronus": "Lontra"},
          {"name": "Harry", "house": "Gryffindor", "patronus": "Cervo"},
          {"name": "Ron", "house": "Gryffindor", "patronus": "Jack Russell Terrier"},
          {"name": "Draco", "house": "Slytherin", "patronus": None},
      ]

  Observe como esse código cria uma `list` de `dict`s. A `list` chamada `students` possui quatro `dicts` dentro dela: um para cada estudante. Além disso, observe que o Python possui uma designação especial `None` quando não há valor associado a uma chave.

- Agora, você tem acesso a uma série de dados interessantes sobre esses estudantes. Agora, modifique ainda mais seu código da seguinte maneira:

      students = [
          {"name": "Hermoine", "house": "Gryffindor", "patronus": "Lontra"},
          {"name": "Harry", "house": "Gryffindor", "patronus": "Cervo"},
          {"name": "Ron", "house": "Gryffindor", "patronus": "Jack Russell Terrier"},
          {"name": "Draco", "house": "Slytherin", "patronus": None},
      ]

      for student in students:
          print(student["name"], student["house"], student["patronus"], sep=", ")

  Observe como o loop `for` irá iterar por cada um dos `dict`s dentro da `list` chamada `students`.

- Você pode aprender mais na Documentação do Python sobre [`dict`s](https://docs.python.org/pt-br/3/tutorial/datastructures.html#dictionaries).

## Mario <a id="mario"></a>

- Lembre-se de que o jogo clássico Mario tem um herói pulando sobre tijolos. Vamos criar uma representação textual deste jogo.

  ![Bloco do Mario](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide21.png "Bloco do Mario")

- Comece a codificar da seguinte forma:

      print("#")
      print("#")
      print("#")

  Observe como estamos copiando e colando o mesmo código repetidamente.

- Considere como poderíamos melhorar o código da seguinte forma:

      for _ in range(3):
          print("#")

  Observe como isso realiza essencialmente o que queremos criar.

- Considere: Seria possível abstrair ainda mais para resolver problemas mais sofisticados posteriormente com este código? Modifique seu código da seguinte forma:

      def main():
          print_column(3)


      def print_column(height):
          for _ in range(height):
              print("#")


      main()

  Observe como nossa coluna pode crescer tanto quanto desejarmos, sem a necessidade de codificação rígida.

- Agora, vamos tentar imprimir uma linha horizontalmente. Modifique seu código da seguinte forma:

      def main():
          print_row(4)


      def print_row(width):
          print("?" * width)


      main()

  Observe como agora temos um código que pode criar blocos da esquerda para a direita.

- Examinando o slide abaixo, observe como o Mario possui tanto linhas quanto colunas de blocos.

  ![Mario Underground](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide24.png "Mario Underground")

- Considere: Como poderíamos implementar tanto linhas quanto colunas em nosso código? Modifique seu código da seguinte forma:

      def main():
          print_square(3)


      def print_square(size):

          # Para cada linha no quadrado
          for i in range(size):

              # Para cada tijolo na linha
              for j in range(size):

                  # Imprime o tijolo
                  print("#", end="")

              # Imprime uma linha em branco
              print()


      main()

  Observe que temos um loop externo que aborda cada linha do quadrado. Em seguida, temos um loop interno que imprime um tijolo em cada linha. Por fim, temos uma instrução `print` que imprime uma linha em branco.

- Podemos abstrair ainda mais nosso código:

      def main():
          print_square(3)


      def print_square(size):
          for i in range(size):
              print_row(size)


      def print_row(width):
          print("#" * width)


      main()
