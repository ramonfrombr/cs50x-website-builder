## Bucles <a id="bucles"></a>

- Básicamente, los bucles son una forma de hacer algo una y otra vez.
- Empieza escribiendo `code cat.py` en la ventana del terminal.
- En el editor de texto, comienza con el siguiente código:

      print("meow")
      print("meow")
      print("meow")

  Al ejecutar este código escribiendo `python cat.py`, notarás que el programa maulla tres veces.

- Al desarrollarte como programador, debes considerar cómo mejorar áreas de tu código donde escribes lo mismo una y otra vez. Imagina si quisieras hacer “meow” 500 veces. ¿Sería lógico escribir esa misma expresión `print("meow")` repetidamente?
- Los bucles te permiten crear un bloque de código que se ejecuta una y otra vez.

## Bucles While <a id="bucles-while"></a>

- El bucle `while` es casi universal en todos los lenguajes de programación.
- Este tipo de bucle repetirá un bloque de código una y otra vez.
- En la ventana del editor de texto, edita tu código de la siguiente manera:

      i = 3
      while i != 0:
          print("miau")

  ¡Observa cómo, aunque este código ejecutará `print("miau")` varias veces, ¡nunca se detendrá! Continuará repitiéndose indefinidamente. Los bucles `while` funcionan preguntando repetidamente si la condición del bucle se ha cumplido. En este caso, el compilador pregunta, "¿`i` no es igual a cero?" Cuando te quedas atrapado en un bucle que se ejecuta infinitamente, puedes presionar `control-c` en tu teclado para salir del bucle.

- Para corregir este bucle infinito, podemos editar el código de la siguiente manera:

      i = 3
      while i != 0:
        print("miau")
        i = i - 1

  Observa que ahora nuestro código se ejecuta correctamente, reduciendo `i` en `1` en cada "iteración" a través del bucle. El término "iteración" tiene un significado especial en la programación. Con iteración nos referimos a un ciclo a través del bucle. La primera iteración es la "0-ésima" iteración a través del bucle. La segunda es la "1ª" iteración. En programación, comenzamos a contar desde 0, luego 1, luego 2.

- Podemos mejorar aún más nuestro código de la siguiente manera:

        i = 1
        while i <= 3:
            print("miau")
            i = i + 1

  Observa que cuando escribimos `i = i + 1` asignamos el valor de `i` de derecha a izquierda. Aquí, comenzamos `i` en uno, como la mayoría de humanos cuentan (1, 2, 3). Si ejecutas el código anterior, verás que hace "miau" tres veces. En programación, es una buena práctica comenzar a contar desde cero.

- Podemos mejorar nuestro código para comenzar a contar desde cero:

      i = 0
      while i < 3:
          print("miau")
          i += 1

  Observa cómo al cambiar el operador a `i < 3` permite que nuestro código funcione como se espera. Comenzamos contando desde 0, e itera a través de nuestro bucle tres veces, produciendo tres "miaus". Además, observa cómo `i += 1` es lo mismo que decir `i = i + 1`.

- Nuestro código en este punto está ilustrado de la siguiente manera: <pre class="mermaid">
flujo de datos TD
A([inicio]) --> B[i = 0]
B --> C{i < 3}
C -- Verdadero --> D["#prr;miau#prr;"]
D --> E[i += 1]
E --> C
C -- Falso --> F([fin])
</pre>

Observa cómo nuestro bucle cuenta `i` hasta, pero no a través de `3`.

## Bucles `for` <a id="for-loops"></a>

- Un bucle `for` es un tipo diferente de bucle.
- Para entender mejor un bucle `for`, es mejor comenzar hablando sobre un nuevo tipo de variable llamado `list` en Python. Como en otras áreas de nuestras vidas, podemos tener una lista de compras, una lista de tareas, etc.
- Un bucle `for` itera a través de una `list` de elementos. Por ejemplo, en la ventana del editor de texto, modifica tu código `cat.py` de la siguiente manera:

      for i in [0, 1, 2]:
          print("meow")

  Observa lo limpio que es este código en comparación con tu código anterior de bucle `while`. En este código, `i` comienza con `0`, maúlla, se le asigna `1`, maúlla, y finalmente, se le asigna `2`, maúlla, y luego termina.

- Aunque este código logra lo que queremos, existen algunas posibilidades para mejorar nuestro código en casos extremos. A primera vista, nuestro código se ve genial. Sin embargo, ¿qué pasaría si quisieras iterar hasta un millón? Es mejor crear un código que pueda funcionar con tales casos extremos. Por lo tanto, podemos mejorar nuestro código de la siguiente manera:

      for i in range(3):
          print("meow")

  Observa cómo `range(3)` devuelve automáticamente tres valores (`0`, `1` y `2`). Este código se ejecutará y producirá el efecto deseado, maullando tres veces.

- Nuestro código puede mejorarse aún más. Observa cómo nunca usamos `i` explícitamente en nuestro código. Es decir, aunque Python necesita `i` como un lugar para almacenar el número de iteración del bucle, nunca lo usamos con otro propósito. En Python, si una variable como esta no tiene otro significado en nuestro código, simplemente podemos representar esta variable como un guion bajo `_`. Por lo tanto, puedes modificar tu código de la siguiente manera:

      for _ in range(3):
          print("meow")

  Observa cómo cambiar `i` por `_` no tiene ningún impacto en el funcionamiento de nuestro programa.

- Nuestro código puede mejorarse aún más. Para explorar las posibilidades dentro de Python, considera el siguiente código:

      print("meow" * 3)

  Observa cómo maullará tres veces, pero el programa producirá `meowmeowmeow` como resultado. Considera: ¿Cómo podrías crear un salto de línea al final de cada maullido?

- De hecho, puedes editar tu código de la siguiente manera:

      print("meow\n" * 3, end="")

  Observa cómo este código produce tres maullidos, cada uno en una línea separada. Al agregar `end=""` y el `\n`, le indicamos al compilador que agregue un salto de línea al final de cada maullido.

## Mejorando con la Entrada del Usuario <a id="mejorando-con-la-entrada-del-usuario"></a>

- Tal vez queramos obtener información del usuario. Podemos utilizar bucles como una forma de validar la entrada del usuario.
- Un paradigma común en Python es utilizar un bucle `while` para validar la entrada del usuario.
- Por ejemplo, intentemos pedir al usuario un número mayor o igual a 0:

      while True:
          n = int(input("¿Cuál es n? "))
          if n < 0:
              continue
          else:
              break

- Observa que hemos introducido dos nuevas palabras clave en Python, `continue` y `break`. `continue` le indica explícitamente a Python que pase a la siguiente iteración de un bucle. `break`, por otro lado, le indica a Python que "salga" del bucle temprano antes de que haya completado todas sus iteraciones. En este caso, continuaremos a la siguiente iteración del bucle cuando `n` es menor que 0, volviendo a pedir al usuario "¿Cuál es n?". Sin embargo, si `n` es mayor o igual a 0, saldremos del bucle con `break` y permitiremos que el resto de nuestro programa se ejecute.
- Resulta que la palabra clave `continue` es redundante en este caso. Podemos mejorar nuestro código de la siguiente manera:

      while True:
          n = int(input("¿Cuál es n? "))
          if n > 0:
              break

      for _ in range(n):
          print("miau")

  Observa cómo este bucle while se ejecutará continuamente hasta que `n` sea mayor que `0`. Cuando `n` es mayor que `0`, el bucle se rompe.

- Aplicando nuestro aprendizaje previo, podemos utilizar funciones para mejorar aún más nuestro código:

      def main():
          miau(obtener_numero())


      def obtener_numero():
          while True:
              n = int(input("¿Cuál es n? "))
              if n > 1:
                  return n


      def miau(n):
          for _ in range(n):
              print("miau")


      main()

  Observa cómo no solo cambiamos tu código para que opere en múltiples funciones, sino que también utilizamos una declaración `return` para devolver el valor de `n` de vuelta a la función `main`.

## Más sobre Listas <a id="más-sobre-listas"></a>

- Considera el mundo de Hogwarts del famoso universo de Harry Potter.
- En la terminal, escribe `code hogwarts.py`.
- En el editor de texto, programa de la siguiente manera:

       estudiantes = ["Hermione", "Harry", "Ron"]

       print(estudiantes[0])
       print(estudiantes[1])
       print(estudiantes[2])

  Observa cómo tenemos una `lista` de estudiantes con sus nombres como se muestra arriba. Luego imprimimos el estudiante que se encuentra en la posición 0, "Hermione". Cada uno de los otros estudiantes también se imprime.

- Tal como ilustramos previamente, podemos usar un bucle para iterar sobre la lista. Puedes mejorar tu código de la siguiente manera:

       estudiantes = ["Hermione", "Harry", "Ron"]

       for estudiante in estudiantes:
           print(estudiante)

  Observa que para cada `estudiante` en la lista de `estudiantes`, imprimirá al estudiante como se pretende. Puede que te preguntes por qué no usamos el guion bajo como se discutió anteriormente. Elegimos no hacerlo porque `estudiante` se utiliza explícitamente en nuestro código.

- Puedes aprender más en la documentación de Python sobre [listas](https://docs.python.org/3/tutorial/datastructures.html#more-on-lists).

## Longitud <a id="longitud"></a>

- Podemos utilizar `len` como una forma de verificar la longitud de la `lista` llamada `students`.
- Imagina que no simplemente quieres imprimir el nombre del estudiante, sino también su posición en la lista. Para lograr esto, puedes editar tu código de la siguiente manera:

      students = ["Hermione", "Harry", "Ron"]

      for i in range(len(students)):
          print(i + 1, students[i])

  Observa cómo ejecutar este código no solo te da la posición de cada estudiante más uno usando `i + 1`, sino que también imprime el nombre de cada estudiante. `len` te permite ver dinámicamente cuánto mide la lista de estudiantes sin importar cuánto crezca.

- Puedes aprender más en la documentación de Python sobre [len](https://docs.python.org/3/library/functions.html?highlight=len#len).

## Diccionarios <a id="dictionaries"></a>

- `dict`s o diccionarios es una estructura de datos que te permite asociar claves con valores.
- Donde una `list` es una lista de múltiples valores, un `dict` asocia una clave con un valor.
- Considerando las casas de Hogwarts, podríamos asignar estudiantes específicos a casas específicas.

  ![Nombres de Harry Potter.](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide17.png "Nombres de Harry Potter.")

- Podríamos usar solo `list` para lograr esto:

      students = ["Hermoine", "Harry", "Ron", "Draco"]
      houses = ["Gryffindor", "Gryffindor", "Griffindor", "Slytherin"]

  Nota que podemos asegurar que siempre mantendremos estas listas en orden. La persona en la primera posición de `students` está asociada con la casa en la primera posición de la lista `houses`, y así sucesivamente. ¡Sin embargo, esto puede volverse bastante engorroso a medida que nuestras listas crecen!

- Podemos mejorar nuestro código usando un `dict` de la siguiente manera:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      print(students["Hermoine"])
      print(students["Harry"])
      print(students["Ron"])
      print(students["Draco"])

  Observa cómo utilizamos llaves `{}` para crear un diccionario. Mientras que las `list`s usan números para iterar a través de la lista, los `dict`s nos permiten usar palabras.

- Ejecuta tu código y asegúrate de que tu salida sea la siguiente:

      $ python hogwarts.py
      Gryffindor
      Gryffindor
      Gryffindor
      Slytherin

- Podemos mejorar nuestro código de la siguiente manera:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      for student in students:
          print(student)

  Observa cómo, al ejecutar este código, el bucle `for` solo iterará a través de todas las claves, resultando en una lista de los nombres de los estudiantes. ¿Cómo podríamos imprimir tanto los valores como las claves?

- Modifica tu código de la siguiente manera:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      for student in students:
          print(student, students[student])

  Observa cómo `students[student]` irá a la clave de cada estudiante y encontrará el valor de su casa. Ejecuta tu código y notarás que la salida es un poco desordenada.

- Podemos limpiar la función de impresión al mejorar nuestro código de la siguiente manera:

      students = {
          "Hermoine": "Gryffindor",
          "Harry": "Gryffindor",
          "Ron": "Gryffindor",
          "Draco": "Slytherin",
      }
      for student in students:
          print(student, students[student], sep=", ")

  Observa cómo esto crea una separación limpia de `,` entre cada ítem impreso.

- Si ejecutas `python hogwarts.py`, deberías ver lo siguiente:

      $ python hogwarts.py
      Hermoine, Gryffindor
      Harry, Gryffindor
      Ron, Gryffindor
      Draco, Slytherin

- ¿Qué pasa si tenemos más información sobre nuestros estudiantes? ¿Cómo podríamos asociar más datos con cada uno de los estudiantes?

  ![Nombres de Harry Potter.](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide18.png "Nombres de Harry Potter.")

- Puedes imaginar que quieres tener mucha información asociada con múltiples claves. Mejora tu código de la siguiente manera:

      students = [
          {"name": "Hermoine", "house": "Gryffindor", "patronus": "Nutria"},
          {"name": "Harry", "house": "Gryffindor", "patronus": "Ciervo"},
          {"name": "Ron", "house": "Gryffindor", "patronus": "Jack Russell terrier"},
          {"name": "Draco", "house": "Slytherin", "patronus": None},
      ]

  Observa cómo este código crea una `list` de `dict`s. La lista llamada `students` tiene cuatro `dicts` dentro de ella: Uno por cada estudiante. Además, nota que Python tiene una designación especial `None` donde no hay un valor asociado con una clave.

- Ahora, tienes acceso a una gran cantidad de datos interesantes sobre estos estudiantes. Ahora, modifica aún más tu código de la siguiente manera:

      students = [
          {"name": "Hermoine", "house": "Gryffindor", "patronus": "Nutria"},
          {"name": "Harry", "house": "Gryffindor", "patronus": "Ciervo"},
          {"name": "Ron", "house": "Gryffindor", "patronus": "Jack Russell terrier"},
          {"name": "Draco", "house": "Slytherin", "patronus": None},
      ]

      for student in students:
          print(student["name"], student["house"], student["patronus"], sep=", ")

  Observa cómo el bucle `for` iterará a través de cada uno de los `dict`s dentro de la `list` llamada `students`.

- Puedes aprender más en la Documentación de Python de los [`dict`s](https://docs.python.org/3/tutorial/datastructures.html#dictionaries).

## Mario <a id="mario"></a>

- Recuerda que el clásico juego de Mario tiene a un héroe saltando sobre ladrillos. Creemos una representación textual de este juego.

  ![Bloque de Mario.](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide21.png "Bloque de Mario.")

- Comienza a codificar de la siguiente manera:

      print("#")
      print("#")
      print("#")

  Observa cómo estamos copiando y pegando el mismo código una y otra vez.

- Considera cómo podríamos mejorar el código de la siguiente manera:

      for _ in range(3):
          print("#")

  Observa cómo esto logra básicamente lo que queremos crear.

- Considera: ¿Podríamos abstraer aún más para resolver problemas más sofisticados más adelante con este código? Modifica tu código de la siguiente manera:

      def main():
          print_column(3)


      def print_column(height):
          for _ in range(height):
              print("#")


      main()

  Observa cómo nuestra columna puede crecer tanto como queramos sin codificación rígida.

- Ahora, intentemos imprimir una fila horizontalmente. Modifica tu código de la siguiente manera:

      def main():
          print_row(4)


      def print_row(width):
          print("?" * width)


      main()

  Observa cómo ahora tenemos un código que puede crear bloques de izquierda a derecha.

- Examinando la diapositiva a continuación, observa cómo Mario tiene tanto filas como columnas de bloques.

  ![Mario Subterráneo.](https://cs50.harvard.edu/python/2022/notes/2/cs50pWeek2Slide24.png "Mario Subterráneo.")

- Considera: ¿Cómo podríamos implementar tanto filas como columnas dentro de nuestro código? Modifica tu código de la siguiente manera:

      def main():
          print_square(3)


      def print_square(size):

          # Para cada fila en el cuadrado
          for i in range(size):

              # Para cada ladrillo en la fila
              for j in range(size):

                  # Imprimir ladrillo
                  print("#", end="")

              # Imprimir línea en blanco
              print()


      main()

  Observa que tenemos un bucle exterior que aborda cada fila en el cuadrado. Luego, tenemos un bucle interno que imprime un ladrillo en cada fila. Finalmente, tenemos una instrucción `print` que imprime una línea en blanco.

- Podemos abstraer aún más nuestro código:

      def main():
          print_square(3)


      def print_square(size):
          for i in range(size):
              print_row(size)


      def print_row(width):
          print("#" * width)


      main()
