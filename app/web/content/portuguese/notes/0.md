## HTML (Linguagem de Marcação Hipertexto)

-   HTML é uma linguagem de marcação que define a estrutura de uma página da web. É interpretada pelo seu navegador da web (Safari, Google Chrome, Firefox, etc.) para exibir conteúdo na tela.
-   Vamos começar escrevendo um arquivo HTML simples!

```html
<!DOCTYPE html>
<html lang="pt">
    <head>
        <title>Hello!</title>
    </head>
    <body>
        Hello, world!
    </body>
</html>
```

-   Quando abrimos esse arquivo em nosso navegador, obtemos:

![Página de Saudação](https://cs50.harvard.edu/web/2020/notes/0/images/hello0.png)

-   Agora, vamos dedicar um tempo para falar sobre o arquivo que acabamos de escrever, que parece ser bastante complicado para uma página tão simples.
    -   Na primeira linha, estamos declarando (para o navegador da web) que estamos escrevendo o documento na versão mais recente do HTML: HTML5.
    -   Em seguida, a página consiste em elementos **HTML aninhados** (como html e body), cada um com uma **tag de abertura e fechamento** marcada com `<element>` para abertura e `</element>` para fechamento.
    -   Observe como cada um dos elementos internos está recuado um pouco mais do que o anterior. Embora isso não seja necessariamente exigido pelo navegador, será muito útil manter este padrão em seu código.
    -   Elementos HTML podem incluir **atributos**, que fornecem informações extras ao navegador sobre o elemento. Por exemplo, ao incluirmos `lang="en"` em nossa tag inicial, estamos informando ao navegador que estamos usando o inglês como idioma principal.
    -   Dentro do elemento HTML, costumamos incluir tanto uma tag `head` quanto uma tag `body`. O elemento head incluirá informações sobre a sua página que não necessariamente são exibidas, e o elemento body conterá o que é realmente visível para os usuários que visitam o site.
    -   Dentro do head, incluímos um `title` para a nossa página da web, que você notará é exibido na guia no topo do navegador da web.
    -   Por fim, incluímos o texto “Hello, world!” no body, que é a parte visível de nossa página.

### Mais Elementos HTML

-   Existem muitos elementos HTML que você pode querer usar para personalizar sua página, incluindo títulos, listas e seções em negrito. No próximo exemplo, veremos alguns destes em ação.
-   Mais uma coisa a ser observada: `<!-- -->` nos dá um comentário em HTML, então usaremos isso abaixo para explicar alguns dos elementos.

```html
<!DOCTYPE html>
<html lang="pt">
    <head>
        <title>Elementos HTML</title>
    </head>
    <body>
        <!-- Podemos criar títulos usando h1 a h6 como tags. -->
        <h1>Um Título Grande</h1>
        <h2>Um Título Menor</h2>
        <h6>O Menor Título</h6>

        <!-- As tags strong e i nos dão texto em negrito e itálico, respectivamente. -->
        Uma palavra <strong>em negrito</strong> e uma palavra <i>em itálico</i>!

        <!-- Podemos vincular a outra página (como a página do cs50) usando a. -->
        Veja o <a href="https://cs50.harvard.edu/">Site do CS50</a>!

        <!-- Usamos ul para uma lista não ordenada e ol para uma ordenada. Ambas listas ordenadas e não ordenadas contêm li, ou itens de lista. -->
        Uma lista não ordenada:
        <ul>
            <li>foo</li>
            <li>bar</li>
            <li>baz</li>
        </ul>
        Uma lista ordenada:
        <ol>
            <li>foo</li>
            <li>bar</li>
            <li>baz</li>
        </ol>

        <!-- Imagens requerem um atributo src, que pode ser o caminho para um arquivo em seu computador ou o link para uma imagem online. Também inclui um atributo alt, que fornece uma descrição caso a imagem não consiga ser carregada. -->
        Uma imagem:
        <img src="../../images/duck.jpeg" alt="Imagem de Pato de Borracha" />
        <!-- Também podemos observar acima que para alguns elementos que não contêm outros, as tags de fechamento não são necessárias. -->

        <!-- Aqui, usamos uma tag br para adicionar espaço em branco à página. -->
        <br />
        <br />

        <!-- Vários tags são necessários para criar uma tabela. -->
        <table>
            <thead>
                <th>Oceano</th>
                <th>Profundidade Média</th>
                <th>Profundidade Máxima</th>
            </thead>
            <tbody>
                <tr>
                    <td>Pacífico</td>
                    <td>4280 m</td>
                    <td>10911 m</td>
                </tr>
                <tr>
                    <td>Atlântico</td>
                    <td>3646 m</td>
                    <td>8486 m</td>
                </tr>
            </tbody>
        </table>
    </body>
</html>
```

### Formulários

-   Outro conjunto de elementos que é realmente importante ao criar um site é como coletar informações dos usuários. Você pode permitir que os usuários insiram informações usando um formulário HTML, que pode conter vários tipos diferentes de entradas. Mais tarde no curso, aprenderemos sobre como lidar com informações uma vez que um formulário tenha sido enviado.
-   Assim como com outros elementos HTML, não há necessidade de memorizá-los, e o W3 Schools é um ótimo recurso para aprender sobre eles!

```html
<!DOCTYPE html>
<html lang="pt">
    <head>
        <title>Formulários</title>
    </head>
    <body>
        <form>
            <input type="text" placeholder="Primeiro Nome" name="first" />
            <input type="password" placeholder="Senha" name="password" />
            <div>
                Cor Favorita:
                <input name="color" type="radio" value="azul" /> Azul
                <input name="color" type="radio" value="verde" /> Verde
                <input name="color" type="radio" value="amarelo" /> Amarelo
                <input name="color" type="radio" value="vermelho" /> Vermelho
            </div>
            <input type="submit" />
        </form>
    </body>
</html>
```

## CSS (Folhas de Estilo em Cascata)

-   CSS é usado para personalizar a aparência de um site.
-   Enquanto estamos começando, podemos adicionar um atributo de estilo a qualquer elemento HTML para aplicar algum CSS a ele.
-   Alteramos o estilo alterando as propriedades CSS de um elemento, escrevendo algo como `color: blue` ou `text-align: center`.
-   No exemplo abaixo, fazemos uma pequena alteração em nosso primeiro arquivo para dar um cabeçalho colorido:

<pre>
<code>&lt;!DOCTYPE html>
&lt;html lang="pt">
    &lt;head>
        &lt;title>Olá!&lt;/title>
    &lt;/head>
    &lt;body>
        &lt;h1 style="color: blue; text-align: center;">Um Cabeçalho Colorido!&lt;/h1>
        Olá, mundo!
    &lt;/body>
&lt;html></code></pre>

![cabeçalho azul](https://cs50.harvard.edu/web/2020/notes/0/images/style0.png)

-   Se aplicarmos estilo a um elemento externo, todos os elementos internos assumem automaticamente esse estilo. Podemos ver isso se movermos o estilo que acabamos de aplicar da tag de cabeçalho para a tag de corpo:

<pre>
<code>&lt;!DOCTYPE html>
&lt;html lang="pt">
    &lt;head>
        &lt;title>Olá!&lt;/title>
    &lt;/head>
    &lt;body style="color: blue; text-align: center;">
        &lt;h1 >Um Cabeçalho Colorido!&lt;/h1>
        Olá, mundo!
    &lt;/body>
&lt;html></code></pre>

![azul em todo lugar](https://cs50.harvard.edu/web/2020/notes/0/images/style1.png)

-   Enquanto podemos estilizar nossa página da web como fizemos acima, para obter um melhor design, devemos ser capazes de separar nosso estilo das linhas individuais.

    -   Uma maneira de fazer isso é adicionar seu estilo entre as tags `<style>` na seção `head`. Dentro dessas tags, escrevemos quais tipos de elementos queremos estilizar e o estilo que desejamos aplicar a eles. Por exemplo:

<pre>
<code>&lt;!DOCTYPE html>
&lt;html lang="pt">
    &lt;head>
        &lt;title>Olá!&lt;/title>
        &lt;style>
            h1 {
                color: blue;
                text-align: center;
            }
        &lt;/style>
    &lt;/head>
    &lt;body>
        &lt;h1 >Um Cabeçalho Colorido!&lt;/h1>
        Olá, mundo!
    &lt;/body>
&lt;/html></code></pre>

-   Outra maneira é incluir em um elemento `<link>` na seção `head` com um link para um arquivo styles.css que contenha algum estilo. Isso significa que o arquivo HTML ficaria assim:

<pre>
<code>&lt;!DOCTYPE html>
&lt;html lang="pt">
&lt;head>
    &lt;title>Olá!&lt;/title>
    &lt;link rel="stylesheet" href="styles.css">
&lt;/head>
&lt;body>
    &lt;h1 >Um Cabeçalho Colorido!&lt;/h1>
    Olá, mundo!
&lt;/body>
&lt;/html></code></pre>

E nosso arquivo chamado `styles.css` ficaria assim:

    h1 {
        color: blue;
        text-align: center;
    }

-   Existem muitas propriedades de CSS para revisar aqui, mas assim como elementos HTML, geralmente é fácil pesquisar algo como “como mudar a cor da fonte para azul com CSS” para obter o resultado. No entanto, algumas das mais comuns são:
    -   `color`: a cor do texto
    -   `text-align`: onde os elementos são posicionados na página
    -   `background-color`: pode ser definido como qualquer cor
    -   `width`: em pixels ou porcentagem de uma página
    -   `height`: em pixels ou porcentagem de uma página
    -   `padding`: quanto espaço deve ser deixado dentro de um elemento
    -   `margin`: quanto espaço deve ser deixado fora de um elemento
    -   `font-family`: tipo de fonte para o texto na página
    -   `font-size`: em pixels
    -   `border`: tamanho tipo (sólido, pontilhado, etc.), cor

<pre>
<code>&lt;!DOCTYPE html>
&lt;html lang="pt">
    &lt;head>
        &lt;title>Tabela Melhor&lt;/title>
    &lt;/head>
    &lt;body>
        &lt;table>
            &lt;thead>
                &lt;th>Oceano&lt;/th>
                &lt;th>Profundidade Média&lt;/th>
                &lt;th>Profundidade Máxima&lt;/th>
            &lt;/thead>
            &lt;tbody>
                &lt;tr>
                    &lt;td>Pacífico&lt;/td>
                    &lt;td>4280 m&lt;/td>
                    &lt;td>10911 m&lt;/td>
                &lt;/tr>
                &lt;tr>
                    &lt;td>Atlântico&lt;/td>
                    &lt;td>3646 m&lt;/td>
                    &lt;td>8486 m&lt;/td>
                &lt;/tr>
            &lt;/tbody>
        &lt;/table>
    &lt;/body>
&lt;html></code></pre>

![tabela ruim](https://cs50.harvard.edu/web/2020/notes/0/images/table0.png)

-   O que vimos acima se parece muito ao que tínhamos anteriormente, mas agora, seja incluindo uma tag `style` ou um `link` para uma folha de estilo na seção `head`, adicionamos o seguinte CSS:

<pre>
<code>table {
    border: 1px solid black;
    border-collapse: collapse;
}

td {
    border: 1px solid black;
    padding: 2px;
}

th {
    border: 1px solid black;
    padding: 2px;
}</code></pre>

O que nos deixa com esta tabela mais bonita:

![tabela boa](https://cs50.harvard.edu/web/2020/notes/0/images/table1.png)

-   Você pode já estar pensando que há alguma repetição desnecessária em nosso CSS no momento, já que `td` e `th` possuem o mesmo estilo. Podemos (e devemos) condensar isso para o seguinte código, usando uma vírgula para mostrar que o estilo deve ser aplicado a mais de um tipo de elemento.

        table {
          border: 1px solid black;
          border-collapse: collapse;
        }

        td, th {
          border: 1px solid black;
          padding: 2px;
        }

-   Isso é uma boa introdução ao que são conhecidos como [seletores de CSS](https://www.w3schools.com/cssref/css_selectors.asp). Existem muitas maneiras de determinar quais elementos HTML você está estilizando, algumas das quais mencionaremos aqui:
    -   **tipo de elemento**: é o que temos feito até agora: estilizando todos os elementos do mesmo tipo.
    -   **id**: Outra opção é atribuir um id aos nossos elementos HTML assim: `<h1 id="primeiro-cabeçalho">Olá!</h1>` e então aplicar estilos usando `#primeiro-cabeçalho {...}` usando a hashtag para mostrar que estamos pesquisando por id. É importante destacar que nenhum elemento pode ter o mesmo id e nenhum elemento pode ter mais de um id.
    -   **classe**: Isso é semelhante ao id, mas uma classe pode ser compartilhada por mais de um elemento, e um único elemento pode ter mais de uma classe. Adicionamos classes a um elemento HTML assim: `<h1 class="texto-página acinzentado">Olá!</h1>` (observe que acabamos de adicionar duas classes ao elemento: `texto-página` e `acinzentado`). Depois estilizamos com base na classe usando um ponto em vez de uma hashtag: `.acinzentado {...}`
-   Agora, também precisamos lidar com o problema de CSS potencialmente conflitante. O que acontece quando um cabeçalho deve ser vermelho com base na classe, mas azul com base no id? O CSS tem uma ordem de especificidade que é a seguinte:
    1.  Estilo inline
    2.  id
    3.  classe
    4.  tipo de elemento
-   Além da vírgula para múltiplos seletores, existem várias outras maneiras de especificar quais elementos você deseja estilizar. Esta tabela da aula fornece algumas informações, e passaremos por alguns exemplos abaixo:

<pre>
<code>&lt;!DOCTYPE html>

    &lt;head>
        &lt;title>Usando Seletores&lt;/title>
        &lt;style>
            ul li {
                color: blue;
            }
        &lt;/style>
    &lt;/head>
    &lt;body>
        &lt;ol>
            &lt;li>foo&lt;/li>
            &lt;li> bar
                &lt;ul>
                    &lt;li>hello&lt;/li>
                    &lt;li>goodbye&lt;/li>
                    &lt;li>hello&lt;/li>
                &lt;/ul>
            &lt;/li>
            &lt;li>baz&lt;/li>
        &lt;/ol>

    &lt;/body>
&lt;html></code></pre>

**Atributos como Seletores**: Também podemos restringir nossa seleção com base nos atributos que atribuímos aos elementos HTML usando colchetes. Por exemplo, na lista de links a seguir, optamos por tornar apenas o link para a Amazon vermelho:

<pre>
<code>&lt;!DOCTYPE html>

    &lt;/head>
    &lt;body>
        &lt;ol>
            &lt;li>&lt;a href="https://www.google.com/">Google&lt;/a>&lt;/li>
            &lt;li>&lt;a href="https://www.amazon.com/">Amazon&lt;/a> &lt;/li>
            &lt;li>&lt;a href="https://www.facebook.com/">Facebook&lt;/a>&lt;/li>
        &lt;/ol>

    &lt;/body>
&lt;html></code></pre>

![seletores de link](https://cs50.harvard.edu/web/2020/notes/0/images/selectors2.png)

-   Não apenas podemos usar o CSS para alterar permanentemente o que um elemento parece, mas também o que ele parece sob determinadas condições. Por exemplo, e se quisermos que um botão mude de cor quando passamos o mouse sobre ele? Podemos fazer isso usando uma [pseudoclasse CSS](https://www.w3schools.com/css/css_pseudo_classes.asp), que fornece estilo adicional sob circunstâncias especiais. Escrevemos isso adicionando dois pontos após nosso seletor e, em seguida, adicionando a circunstância após esse dois pontos.
-   No caso do botão, acrescentaríamos `:hover` ao seletor do botão para especificar o design apenas ao passar o mouse sobre:

<pre>

            }

            }
        &lt;/style>
    &lt;/head>
    &lt;body>
        &lt;button>Botão 1&lt;/button>
        &lt;button>Botão 2&lt;/button>
        &lt;button>Botão 3&lt;/button>

    &lt;/body>
&lt;html></code></pre>

## Design Responsivo

-   Hoje, muitas pessoas visualizam sites em dispositivos diferentes de computadores, como smartphones e tablets. É importante garantir que seu site seja legível para pessoas em todos os dispositivos.
-   Uma maneira de conseguirmos isso é por meio do conhecimento do **viewport**. O viewport é a parte da tela que é visível para o usuário a qualquer momento. Por padrão, muitas páginas da web assumem que o viewport é o mesmo em qualquer dispositivo, o que leva muitos sites (principalmente os mais antigos) a serem difíceis de interagir em dispositivos móveis.
-   Uma maneira simples de melhorar a aparência de um site em um dispositivo móvel é adicionar a seguinte linha no cabeçalho de nossos arquivos HTML. Essa linha instrui o dispositivo móvel a usar um viewport com a mesma largura do dispositivo que você está usando, em vez de um muito maior.

```
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

-   Outra maneira de lidarmos com diferentes dispositivos é por meio de [media queries](https://www.w3schools.com/cssref/css3_pr_mediaquery.asp). As media queries são formas de alterar o estilo de uma página com base em como a página está sendo visualizada.
-   Para um exemplo de uma media query, vamos tentar simplesmente alterar a cor da tela quando ela é reduzida para um determinado tamanho. Sinalizamos uma media query digitando `@media`, seguido pelo tipo de consulta entre parênteses:

```
<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <title>Tamanho da Tela</title>
        <style>
            @media (min-width: 600px) {
                body {
                    background-color: red;
                }
            }

            @media (max-width: 599px) {
                body {
                    background-color: blue;
                }
            }
        </style>
    </head>
    <body>
        <h1>Bem-vindo à página!</h1>
    </body>
</html>
```

![tamanho da tela](https://cs50.harvard.edu/web/2020/notes/0/images/responsive0.gif)

-   Outra maneira de lidar com tamanhos de tela diferentes é usando um novo atributo CSS conhecido como [flexbox](https://www.w3schools.com/css/css3_flexbox.asp). Isso nos permite facilmente fazer com que os elementos sejam transferidos para a próxima linha se não couberem horizontalmente. Fazemos isso colocando todos os nossos elementos em uma `div` que chamaremos de nosso container. Em seguida, adicionamos alguns estilos a essa div especificando que queremos usar um display flexbox para os elementos dentro dela. Também adicionamos alguns estilos adicionais às divs internas para ilustrar melhor o embrulho que está ocorrendo aqui.

```
<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <title>Tamanho da Tela</title>
        <style>
            #container {
                display: flex;
                flex-wrap: wrap;
            }

            #container > div {
                background-color: green;
                font-size: 20px;
                margin: 20px;
                padding: 20px;
                width: 200px;
            }
        </style>
    </head>
    <body>
        <div id="container">
            <div>Texto 1!</div>
            <div>Texto 2!</div>
            <div>Texto 3!</div>
            <div>Texto 4!</div>
            <div>Texto 5!</div>
            <div>Texto 6!</div>
            <div>Texto 7!</div>
            <div>Texto 8!</div>
            <div>Texto 9!</div>
            <div>Texto 10!</div>
            <div>Texto 11!</div>
            <div>Texto 12!</div>
        </div>
    </body>
</html>
```

## Bootstrap

-   Acontece que existem muitas bibliotecas que outras pessoas já escreveram e que podem tornar a estilização de uma página da web ainda mais simples. Uma biblioteca popular que usaremos ao longo do curso é conhecida como [bootstrap](https://getbootstrap.com/).
-   Podemos incluir o bootstrap em nosso código adicionando uma única linha no cabeçalho do nosso arquivo HTML:

```html
<link
    rel="stylesheet"
    href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.0/css/bootstrap.min.css"
    integrity="sha384-9aIt2nRpC12Uk9gS9baDl411NQApFmC26EwAOH8WgZl5MYYxFfc+NcPb1dKGj7Sk"
    crossorigin="anonymous"
/>
```

-   Em seguida, podemos explorar algumas das características do bootstrap navegando até a parte de [documentação](https://getbootstrap.com/docs/4.5/components/) do site deles. Nesta página, você encontrará muitos exemplos de classes que você pode adicionar aos elementos que permitem a estilização com bootstrap.
-   Uma característica popular do bootstrap é o seu [sistema de grade](https://getbootstrap.com/docs/4.0/layout/grid/). ## Sass (Folhas de Estilo Incríveis e Sintáticas)

-   Até agora, encontramos algumas maneiras de eliminar a redundância no CSS, como movendo-o para arquivos separados ou usando o bootstrap, mas ainda existem várias áreas onde podemos fazer melhorias. Por exemplo, e se quisermos que vários elementos tenham estilos diferentes, mas todos eles tenham a mesma cor? Se decidirmos posteriormente mudar a cor, teríamos que fazer essa alteração em vários elementos diferentes.
-   [Sass](https://sass-lang.com/) é uma linguagem que nos permite escrever CSS de maneira mais eficiente de várias formas, uma das quais é permitindo que usemos variáveis, como no exemplo a seguir.
-   Ao escrever em Sass, criamos um novo arquivo com a extensão `filename.scss`. Neste arquivo, podemos criar uma nova variável adicionando um `$` antes de um nome, seguido de dois pontos e um valor. Por exemplo, escreveríamos `$color: red` para definir a variável de cor com o valor vermelho. Em seguida, acessamos essa variável usando `$color`. Aqui está um exemplo de nosso arquivo variables.scss:

        $color: red;

        ul {
          font-size: 14px;
          color: $color;
        }

        ol {
          font-size: 18px;
          color: $color;
        }

-   Agora, para vincular esse estilo ao nosso arquivo HTML, não podemos simplesmente vincular o arquivo `.scss` porque a maioria dos navegadores reconhece apenas arquivos `.css`. Para lidar com esse problema, precisamos [baixar um programa chamado Sass](https://sass-lang.com/install) em nossos computadores. Em seguida, em nosso terminal, digitamos `sass variables.scss:variables.css`. Este comando compilará um arquivo .scss chamado `variables.scss` em um arquivo .css chamado `variables.css`, ao qual você pode adicionar um link em sua página HTML.
-   Para acelerar esse processo, podemos usar o comando `sass --watch variables.scss:variables.css`, que muda automaticamente o arquivo `.css` sempre que uma alteração é detectada no arquivo `.scss`.
-   Ao usar Sass, também podemos aninhar o nosso estilo fisicamente em vez de usar os seletores CSS mencionados anteriormente. Por exemplo, se quisermos aplicar alguns estilos apenas a parágrafos e listas não ordenadas dentro de uma div, podemos escrever o seguinte:

        div {
          font-size: 18px;

          p {
            color: blue;
          }

          ul {
            color: green;
          }
        }

Uma vez compilado em CSS, teríamos um arquivo que se parece com:

    div {
        font-size: 18px;
    }

    div p {
        color: blue;
    }

    div ul {
        color: green;
    }

-   Mais uma funcionalidade que o Sass nos oferece é conhecida como [herança](https://sass-lang.com/guide). Isso nos permite criar um conjunto básico de estilos que pode ser compartilhado por vários elementos diferentes. Fazemos isso adicionando um `%` antes do nome de uma classe, adicionando alguns estilos e, posteriormente, adicionando a linha `@extend %nomeDaClasse` no início de alguns estilos. Por exemplo, o código a seguir aplica o estilo da classe `message` a cada uma das diferentes classes abaixo, resultando em uma página da web que se parece com a seguinte.

        %message {
          font-family: sans-serif;
          font-size: 18px;
          font-weight: bold;
          border: 1px solid black;
          padding: 20px;
          margin: 20px;
        }

        .success {
          @extend %message;
          background-color: green;
        }

        .warning {
          @extend %message;
          background-color: orange;
        }

        .error {
          @extend %message;
          background-color: red;
        }
