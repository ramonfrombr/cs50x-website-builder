O que Fazer

  1. Envie este formulário.
  2. Visite este link, faça login com sua conta do GitHub e clique em Autorizar CS50.
  3. Marque a caixa indicando que você gostaria de conceder acesso à equipe do curso à sua submissão e clique em Participar do curso.
  4. Faça login em cs50.dev, que é uma versão baseada em nuvem do Visual Studio Code (VS Code) que fornece seu próprio “codespace” com tudo o que você precisa para o curso pré-instalado. Não é necessário baixar e instalar o VS Code ou Python em seu próprio Mac ou PC! Se ainda não estiver familiarizado, certifique-se de assistir ao Vídeo Curto sobre Visual Studio Code para CS50 para uma visão geral completa.
  5. Assim que seu codespace for carregado, feche quaisquer guias de Bem-vindo que possam ter sido abertas por padrão.
  6. Execute update50 na janela do terminal do seu codespace para garantir que seu codespace esteja atualizado; se solicitado, clique em Reconstruir agora.
  7. Envie todos os problemas abaixo:
    • Voz Interna
    • Velocidade de Reprodução
    • Criando Faces
    • Einstein
    • Calculadora de Gorjeta
