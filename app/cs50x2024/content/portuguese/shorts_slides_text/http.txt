Protocolo de Transferência de Hipertexto (HTTP)
• Além dos protocolos que ditam como as informações são comunicadas de máquina para máquina e de aplicativo para aplicativo (IP e TCP, respectivamente), é frequente que o próprio aplicativo tenha um sistema de regras sobre como interpretar os dados que foram enviados.
• HTTP é um exemplo de um protocolo de camada de aplicativo que dita especificamente o formato pelo qual os clientes solicitam páginas da web a um servidor e o formato pelo qual os servidores devolvem informações aos clientes.

Protocolo de Transferência de Hipertexto (HTTP)
• O nome do host (nome de domínio do servidor) também é incluído como uma linha separada da solicitação geral de HTTP.
• Juntos, o nome do host e o destino da solicitação da linha de solicitação especificam um recurso específico que está sendo buscado.
• Com base em se o recurso existe e se o servidor está capacitado a fornecer esse recurso de acordo com a solicitação do cliente, uma série de códigos de status pode resultar.
