Argumentos de Linha de Comando
● argc (contagem de argumentos)
● Esta variável do tipo inteiro irá armazenar o número de argumentos de linha de comando que o usuário digitou quando o
programa foi executado.
comando
argc
./greedy
1
./greedy 1024 cs50
3

[4]

Argumentos de Linha de Comando
● argv (vetor de argumentos)
● Este array de strings armazena, uma string por elemento, o texto real que o usuário digitou na linha de comando quando
o programa foi executado.
● O primeiro elemento de argv é sempre encontrado em argv[0]. O último elemento de argv é sempre encontrado em
argv[argc-1].
