O que fazer

  1. Faça login no cs50.dev usando sua conta do GitHub
  2. Execute update50 na janela do terminal do seu codespace para garantir que seu codespace esteja atualizado.
  3. Envie Hello em Python
  4. Envie um dos seguintes:
    • esta versão do Mario em Python, se estiver se sentindo menos confortável
    • esta versão do Mario em Python, se estiver se sentindo mais confortável
  5. Envie um dos seguintes:
    • Cash em Python, se estiver se sentindo menos confortável
    • Credit em Python, se estiver se sentindo mais confortável

  7. Envie DNA em Python

Se você enviar as duas versões de Mario, registraremos a maior das duas pontuações. Se você enviar Cash e Credit, registraremos a maior das duas pontuações.

    • Recorrer à web ou a outro lugar para obter instruções além das do próprio curso, referências e soluções para dificuldades técnicas, mas não para soluções diretas para problemas do conjunto de problemas ou para seu próprio projeto final.
