# Certificado CS50

Si envías y recibes una puntuación de al menos el 70% en cada uno de los problemas de este curso, así como en su proyecto final, serás elegible para obtener un [Certificado CS50](https://cs50.harvard.edu/certificates) gratuito como el que se muestra a continuación. Para obtener un [certificado verificado](https://www.edx.org/verified-certificate) de edX, regístrate en [cs50.edx.org/python](http://cs50.edx.org/python). Envía un correo electrónico a [certificates@cs50.harvard.edu](mailto:certificates@cs50.harvard.edu) si tienes alguna pregunta.
