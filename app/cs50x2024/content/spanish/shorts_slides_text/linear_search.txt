• Escenario de peor caso: tenemos que examinar todo el array de n elementos, ya sea porque el elemento objetivo es el último elemento del array o no existe en el array.
• Escenario de mejor caso: el elemento objetivo es el primer elemento del array y, por lo tanto, podemos dejar de buscar inmediatamente después de comenzar.
