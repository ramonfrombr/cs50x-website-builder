Modelo de Objeto de Documento
• Como vimos, os objetos do JavaScript são incrivelmente flexíveis e podem conter vários campos, mesmo quando esses campos são outros objetos.
• O objeto documento é uma maneira de empregar esse paradigma, por meio do qual esse objeto organiza todo o conteúdo de uma página da Web.
• Ao organizar uma página inteira em um objeto JavaScript, podemos manipular os elementos da página programaticamente.

Modelo de Objeto de Documento
• O objeto documento em si, assim como todos os objetos nele contidos, têm várias propriedades e vários métodos que podem ser usados para detalhar uma parte muito específica de seu site.
• Ao redefinir essas propriedades ou chamar determinados métodos, o conteúdo de nossas páginas da Web pode mudar sem que precisemos atualizar a página.

Descrição

• jQuery
• Como a manipulação de DOM é tão comum no JavaScript e como o JavaScript para fazer isso pode ficar bem extenso, as pessoas queriam alternativas.
• O jQuery é uma biblioteca de código aberto popular, lançada em 2006, projetada para simplificar a criação de scripts do lado do cliente (como manipulações de DOM).
