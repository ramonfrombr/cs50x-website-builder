Argumentos de línea de comandos
● argc (conteo de argumentos)
● Esta variable de tipo entero almacenará el número de argumentos de línea de comandos que escribió el usuario cuando se ejecutó el programa.

argc

[4]

Argumentos de línea de comandos
● argv (vector de argumentos)
● Esta matriz de cadenas almacena, una cadena por elemento, el texto real que escribió el usuario en la línea de comandos cuando se ejecutó el programa.
● El primer elemento de argv siempre se encuentra en argv[0]. El último elemento de argv siempre se encuentra en argv[argc-1].
