## Entrada e Saída de Arquivos <a id="file-io"></a>

- Até agora, tudo o que programamos armazenou informações na memória. Ou seja, uma vez que o programa é encerrado, todas as informações coletadas do usuário ou geradas pelo programa são perdidas.
- Entrada e saída de arquivos é a capacidade de um programa de receber um arquivo como entrada ou criar um arquivo como saída.
- Para começar, no terminal, digite `code names.py` e escreva o seguinte código:

      nome = input("Qual é o seu nome?" )
      print(f"Olá, {nome}")

  Observe que a execução deste código tem a saída desejada. O usuário pode digitar um nome. A saída é como esperado.

- No entanto, e se quisermos permitir que vários nomes sejam inseridos? Como podemos fazer isso? Lembre-se de que uma `list` é uma estrutura de dados que nos permite armazenar vários valores em uma única variável. Escreva o seguinte código:

      nomes = []

      for _ in range(3):
          nome = input("Qual é o seu nome?" )
          nomes.append(nome)

  Observe que o usuário será solicitado três vezes a entrada. O método `append` é usado para adicionar o `nome` à nossa lista `nomes`.

- Esse código pode ser simplificado para o seguinte:

      nomes = []

      for _ in range(3):
          nomes.append(input("Qual é o seu nome?" ))

  Observe que isso tem o mesmo resultado que o bloco de código anterior.

  - Agora, vamos permitir a impressão da lista de nomes em ordem alfabética. Escreva o seguinte código:

    nomes = []

    for \_ in range(3):
    nomes.append(input("Qual é o seu nome?" ))

    for nome in sorted(nomes):
    print(f"Olá, {nome}")

  Observe que uma vez que este programa é executado, todas as informações são perdidas. A entrada e saída de arquivos permite que seu programa armazene essas informações para que possam ser usadas posteriormente.

- Você pode aprender mais na documentação do Python sobre [sorted](https://docs.python.org/3/library/functions.html#sorted).

## `open` <a id="open"></a>

- `open` é uma funcionalidade incorporada ao Python que permite abrir um arquivo e utilizá-lo em seu programa. A função `open` permite abrir um arquivo de forma que você possa lê-lo ou escrever nele.
- Para mostrar como habilitar a entrada e saída de arquivo em seu programa, vamos voltar um pouco e codificar da seguinte forma:

      name = input("Qual é o seu nome? ")

      file = open("names.txt", "w")
      file.write(name)
      file.close()

  Observe que a função `open` abre um arquivo chamado `names.txt` com a escrita habilitada, como indicado pelo `w`. O código acima atribui esse arquivo aberto a uma variável chamada `file`. A linha `file.write(name)` escreve o nome no arquivo de texto. A linha posterior fecha o arquivo.

- Testando o código digitando `python names.py`, você pode inserir um nome e ele é salvo no arquivo de texto. No entanto, se você executar seu programa várias vezes usando nomes diferentes, perceberá que este programa reescreverá completamente o arquivo `names.txt` a cada vez.
- Idealmente, queremos ser capazes de acrescentar cada um dos nomes ao arquivo. Remova o arquivo de texto existente digitando `rm names.txt` na janela do terminal. Em seguida, modifique seu código da seguinte forma:

      name = input("Qual é o seu nome? ")

      file = open("names.txt", "a")
      file.write(name)
      file.close()

  Observe que a única alteração em nosso código é que o `w` foi alterado para `a` para "append" (acrescentar). Executando este programa várias vezes, você perceberá que os nomes serão adicionados ao arquivo. No entanto, você perceberá um novo problema!

- Examinando seu arquivo de texto após executar seu programa várias vezes, você verá que os nomes estão se juntando. Os nomes estão sendo adicionados sem espaços entre cada um deles. Você pode resolver esse problema. Novamente, remova o arquivo de texto existente digitando `rm names.txt` na janela do terminal. Em seguida, modifique seu código da seguinte forma:

      name = input("Qual é o seu nome? ")

      file = open("names.txt", "a")
      file.write(f"{name}\n")
      file.close()

  Observe que a linha com `file.write` foi modificada para adicionar uma quebra de linha no final de cada nome.

- Este código está funcionando muito bem. No entanto, existem maneiras de melhorar este programa. Acontece que é fácil esquecer de fechar o arquivo.
- Você pode obter mais informações na documentação do Python sobre [open](https://docs.python.org/3/library/functions.html#open).

## `with` <a id="with"></a>

- A palavra-chave `with` permite automatizar o fechamento de um arquivo.
- Modifique seu código da seguinte forma:

      name = input("Qual é o seu nome? ")

      with open("nomes.txt", "a") as file:
          file.write(f"{name}\n")

  Note que a linha abaixo do `with` está indentada.

- Até este ponto, estivemos exclusivamente escrevendo em um arquivo. E se quisermos ler de um arquivo? Para habilitar essa funcionalidade, modifique seu código da seguinte forma:

      with open("nomes.txt", "r") as file:
          lines = file.readlines()

      for line in lines:
          print("Olá,", line)

  Note que `readlines` possui a habilidade especial de ler todas as linhas de um arquivo e armazená-las em uma variável chamada `lines`. Ao executar o programa, você notará que a saída é um tanto confusa. Parece haver múltiplas quebras de linha onde deveria haver apenas uma.

- Existem várias abordagens para resolver esse problema. No entanto, aqui está uma maneira simples de corrigir esse erro em nosso código:

      with open("nomes.txt", "r") as file:
          lines = file.readlines()

      for line in lines:
          print("Olá,", line.rstrip())

  Note que `rstrip` tem o efeito de remover a quebra de linha excedente no final de cada linha.

- Ainda assim, esse código poderia ser simplificado ainda mais:

      with open("nomes.txt", "r") as file:
          for line in file:
              print("Olá,", line.rstrip())

  Note que ao executar este código, está correto. Entretanto, repare que não estamos ordenando os nomes.

- Este código poderia ser melhorado para permitir a ordenação dos nomes:

      nomes = []

      with open("nomes.txt") as file:
          for line in file:
              nomes.append(line.rstrip())

      for nome in sorted(nomes):
          print(f"Olá, {nome}")

  Note que `nomes` é uma lista vazia na qual podemos coletar os nomes. Cada nome é adicionado à lista `nomes` na memória. Então, cada nome na lista ordenada na memória é impresso. Ao executar seu código, você verá que os nomes estão agora corretamente ordenados.

## CSV <a id="csv"></a>

- CSV significa "valores separados por vírgulas".
- Na sua janela do terminal, digite `code students.csv`. Verifique se seu novo arquivo CSV possui a seguinte aparência:

      Hermione, Grifinória
      Harry, Grifinória
      Ron, Grifinória
      Draco, Sonserina

- Vamos criar um novo programa digitando `code students.py` e código como segue:

      with open("students.csv") as file:
          for line in file:
              row = line.rstrip().split(",")
              print(f"{row[0]} está em {row[1]}")

  Observe que `rstrip` remove o final de cada linha do nosso arquivo CSV. `split` informa ao compilador onde encontrar o final de cada um dos nossos valores no arquivo CSV. `row[0]` é o primeiro elemento em cada linha do nosso arquivo CSV. `row[1]` é o segundo elemento em cada linha no nosso arquivo CSV.

- O código acima é eficaz ao dividir cada linha ou "registro" do nosso arquivo CSV. No entanto, é um pouco criptografado se você não estiver familiarizado com esse tipo de sintaxe. O Python possui uma capacidade incorporada que pode simplificar ainda mais este código. Modifique seu código da seguinte maneira:

      with open("students.csv") as file:
          for line in file:
              name, house = line.rstrip().split(",")
              print(f"{name} está em {house}")

  Observe que a função `split` na verdade retorna dois valores: o que está antes da vírgula e o que está depois da vírgula. Consequentemente, podemos contar com essa funcionalidade para atribuir duas variáveis de uma vez em vez de apenas uma!

- Imagine que gostaríamos novamente de fornecer esta lista como uma saída ordenada. Você pode modificar seu código da seguinte maneira:

      students = []

      with open("students.csv") as file:
          for line in file:
              name, house = line.rstrip().split(",")
              students.append(f"{name} está em {house}")

      for student in sorted(students):
          print(student)

  Observe que criamos uma `lista` chamada `students`. Nós `append` cada string nesta lista. Em seguida, imprimimos uma versão ordenada de nossa lista.

- Lembre-se que o Python permite `dicionários` onde uma chave pode estar associada a um valor. Esse código pode ser melhorado ainda mais

      students = []

      with open("students.csv") as file:
          for line in file:
              name, house = line.rstrip().split(",")
              student = {}
              student["name"] = name
              student["house"] = house
              students.append(student)

      for student in students:
          print(f"{student['name']} está em {student['house']}")

  Observe que criamos um dicionário vazio chamado `student`. Adicionamos os valores de cada aluno, incluindo seu nome e casa no dicionário `student`. Em seguida, adicionamos esse aluno à `lista` chamada `students`.

- Podemos melhorar o código da seguinte maneira:

      students = []

      with open("students.csv") as file:
          for line in file:
              name, house = line.rstrip().split(",")
              student = {"name": name, "house": house}
              students.append(student)

      for student in students:
          print(f"{student['name']} está em {student['house']}")

  Observe que isso produz o resultado desejado, mas sem a ordenação dos alunos.

- Infelizmente, não podemos ordenar os alunos como antes, porque cada aluno é agora um dicionário dentro de uma lista. Seria útil se o Python pudesse classificar a lista `students` de dicionários `student` que classifica essa lista de dicionários pelo nome do aluno.
- Para implementar isso em nosso código, faça as seguintes alterações:

      students = []

      with open("students.csv") as file:
          for line in file:
              name, house = line.rstrip().split(",")
              students.append({"name": name, "house": house})


      def get_name(student):
          return student["name"]


      for student in sorted(students, key=get_name):
          print(f"{student['name']} está em {student['house']}")

  Observe que `sorted` precisa saber como obter a chave de cada aluno. O Python permite um parâmetro chamado `key`, onde podemos definir em qual "chave" a lista de alunos será ordenada. Portanto, a função `get_name` retorna simplesmente a chave `student["name"]`. Executando este programa, você verá que agora a lista está ordenada pelo nome.

- Ainda assim, nosso código pode ser melhorado. Acontece que, se você só vai usar uma função como `get_name` uma vez, você pode simplificar seu código da maneira apresentada abaixo. Modifique seu código da seguinte maneira:

      students = []

      with open("students.csv") as file:
          for line in file:
              name, house = line.rstrip().split(",")
              students.append({"name": name, "house": house})

      for student in sorted(students, key=lambda student: student["name"]):
          print(f"{student['name']} está em {student['house']}")

  Observe como usamos uma função `lambda`, uma função anônima, que diz "Ei Python, aqui está uma função que não tem nome: dado um `student`, acesse seu `name` e retorne isso para a `key`.

- Infelizmente, nosso código é um pouco frágil. Suponha que alteramos nosso arquivo CSV para indicar onde cada aluno cresceu. Qual seria o impacto disso em nosso programa? Primeiro, modifique seu arquivo `students.csv` da seguinte maneira:

  Harry, "Número Quatro, Rua dos Alfeneiros"
  Ron, "A Toca"
  Draco, "Mansão Malfoy"

Observe como executar nosso programa agora produzirá vários erros.

- Agora que estamos lidando com locais de moradia em vez de casas, modifique seu código da seguinte maneira:

      students = []

      with open("students.csv") as file:
          for line in file:
              name, home = line.rstrip().split(",")
              students.append({"name": name, "home": home})

      for student in sorted(students, key=lambda student: student["name"]):
          print(f"{student['name']} está em {student['home']}")

  Observe que a execução do nosso programa ainda não funciona corretamente. Você consegue adivinhar por quê?

- O erro `ValueError: too many values to unpack` produzido pelo compilador é resultado do fato de que anteriormente criamos esse programa esperando que o arquivo CSV seja `split` usando uma `,` (vírgula). Poderíamos passar mais tempo lidando com isso, mas alguém já desenvolveu uma maneira de "analisar" (ou seja, ler) arquivos CSV!
- A biblioteca interna `csv` do Python vem com um objeto chamado `reader`. Como o nome sugere, podemos usar um `reader` para ler nosso arquivo CSV, apesar da vírgula extra em "Número Quatro, Rua dos Alfeneiros". Um `reader` funciona em um loop `for`, onde a cada iteração o `reader` nos fornece outra linha de nosso arquivo CSV. Essa linha em si é uma lista, onde cada valor na lista corresponde a um elemento nessa linha. `row[0]`, por exemplo, é o primeiro elemento da linha fornecida, enquanto `row[1]` é o segundo elemento.

      import csv

      students = []

      with open("students.csv") as file:
          reader = csv.reader(file)
          for row in reader:
              students.append({"name": row[0], "home": row[1]})

      for student in sorted(students, key=lambda student: student["name"]):
          print(f"{student['name']} é de {student['home']}")

  Observe que nosso programa agora funciona conforme o esperado.

## Arquivos Binários e `PIL` <a id="binary-files-and-pil"></a>

- Um outro tipo de arquivo que iremos discutir hoje é um arquivo binário. Um arquivo binário é simplesmente uma coleção de uns e zeros. Esse tipo de arquivo pode armazenar qualquer coisa, incluindo música e dados de imagens.
- Existe uma biblioteca popular em Python chamada `PIL` que funciona bem com arquivos de imagem.
- Os GIFs animados são um tipo popular de arquivo de imagem que contém vários arquivos de imagem dentro dele que são reproduzidos em sequência repetidamente, criando uma animação simplista ou efeito de vídeo.
- Imagine que temos uma série de fantasias, como ilustrado abaixo.
- Aqui está `costume1.gif`.

![Gato número 1.](https://cdn.cs50.net/python/2022/x/lectures/6/src6/costume1.gif "costume1.gif")

- Aqui está outro chamado `costume2.gif`. Observe como as posições das pernas são ligeiramente diferentes.

![Gato número 2.](https://cdn.cs50.net/python/2022/x/lectures/6/src6/costume2.gif "costume2.gif")

- Antes de prosseguir, certifique-se de ter baixado os arquivos de código-fonte do site do curso. Não será possível codificar o seguinte sem possuir as duas imagens acima em sua posse e armazenadas em seu IDE.
- Na janela do terminal, digite `code costumes.py` e codifique da seguinte forma:

      import sys

      from PIL import Image

      images = []

      for arg in sys.argv[1:]:
          image = Image.open(arg)
          images.append(image)

      images[0].save(
          "costumes.gif", save_all=True, append_images=[images[1]], duration=200, loop=0
      )

  Observe que importamos a funcionalidade `Image` do `PIL`. Observe que o primeiro loop `for` simplesmente percorre as imagens fornecidas como argumentos de linha de comando e as armazena na lista chamada `images`. O `1:` começa a fatiar `argv` em seu segundo elemento. As últimas linhas de código salvam a primeira imagem e também acrescentam uma segunda imagem a ela, criando um gif animado. Digite `python costumes.py costume1.gif costume2.gif` no terminal. Agora, digite `code costumes.gif` na janela do terminal e você poderá ver um GIF animado.

- Você pode saber mais na documentação do Pillow sobre [PIL](https://pillow.readthedocs.io/).
