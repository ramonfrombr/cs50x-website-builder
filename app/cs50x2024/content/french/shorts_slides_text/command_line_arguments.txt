● Cette variable de type entier stockera le nombre d'arguments de ligne de commande que l'utilisateur a saisis lorsque le 
programme a été exécuté.

Arguments de ligne de commande
● argv (vecteur d'arguments)
● Ce tableau de chaînes stocke, une chaîne par élément, le texte réel que l'utilisateur a saisi sur la ligne de commande lorsque 
le programme a été exécuté.
● Le premier élément d'argv se trouve toujours à argv[0]. Le dernier élément d'argv se trouve toujours à 
argv[argc-1].
